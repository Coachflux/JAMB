Quick Overview

JAMB Oral English tests your ability to recognize words that rhyme (same ending sound) and homophones (words that sound the same but have different meanings/spellings).

Common question types:

  • “Choose the word that rhymes with the underlined word”
  • “Which pair of words are homophones?”
  • “Select the option with the same sound as…”

Step-by-step learning method:

  1. Read the rhyme set aloud slowly — feel the identical ending sound
  2. Note homophones (words that sound exactly the same)
  3. Practise saying them in sentences to hear differences
  4. Test yourself with past JAMB questions (many repeat these sets)

Quick JAMB Revision Tips for Rhymes & Homophones

  • Rhyme focus: JAMB mostly tests perfect rhymes (same vowel + ending consonant)
  • Homophone trap: Very common — always check meaning/spelling, not just sound
  • Practise aloud: Say each set slowly — feel the identical ending sound
  • Past questions: 2018–2025 repeat sets like -ate, -ight, -ee, -air, -ook, -all
  • Minimal pairs: Practice hear/here, right/write, there/their/they're daily
