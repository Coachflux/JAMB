Binary Operations Made Simple (JAMB Syllabus)

Abumchukwu, a binary operation is a rule (denoted by *, ⊕, Δ, etc.) that takes two elements from a set and produces exactly one result that must also belong to the same set. Think of it like addition (+) or multiplication (×) on real numbers — you always get another real number back. I will teach you step by step like you're in SS1, starting from the basics. We'll look at the properties one by one, then identity and inverse, and solve examples slowly with every step explained.

Part (a): The Four Key Properties

These are the main things JAMB asks about. We'll explain each one, then check with examples.

1. Closure

Step 1: The result of a * b must stay inside the given set for every a and b in the set.
Step 2: If even one pair gives a result outside the set, it's not closed.
Think: Like adding positive numbers — you always get positive, so closed. But subtracting positive numbers can give negative, so not closed on positives.

2. Commutativity (order does not matter)

Step 1: Check if a * b = b * a for all a, b in the set.
Step 2: If it holds for all, it's commutative; if not for even one pair, it's not.
Think: Addition 2 + 3 = 3 + 2 = 5, so commutative. Subtraction 3 – 2 = 1, 2 – 3 = –1, not equal, so not commutative.

3. Associativity (grouping does not matter)

Step 1: Check if (a * b) * c = a * (b * c) for all a, b, c.
Step 2: If it holds for all, associative; if not, not.
Think: Addition (2 + 3) + 4 = 5 + 4 = 9, 2 + (3 + 4) = 2 + 7 = 9, associative. Division (8 / 4) / 2 = 2 / 2 = 1, 8 / (4 / 2) = 8 / 2 = 4, not associative.

4. Distributivity (one operation spreads over another)

Step 1: Usually * distributes over + : a * (b + c) = (a * b) + (a * c) and (b + c) * a = (b * a) + (c * a).
Step 2: Check if both hold for all a, b, c.
Think: Multiplication distributes over addition: 2 × (3 + 4) = 2×7 = 14, (2×3) + (2×4) = 6 + 8 = 14, yes. Addition does not distribute over multiplication: 2 + (3 × 4) = 2 + 12 = 14, (2 + 3) × (2 + 4) = 5×6 = 30, no.

Part (b): Identity and Inverse Elements (Simple Cases Only)

Identity element (e): The “do-nothing” element. Step 1: Solve a * e = e * a = a for all a. The e must be the same for all a.
Inverse element: The “undo” element. Step 1: For any a, solve a * b = b * a = e for b (the inverse of a).
Think: For + , identity is 0 (a + 0 = a). Inverse of 5 is –5 (5 + –5 = 0).
For × , identity is 1 (a × 1 = a). Inverse of 5 is 1/5 (5 × 1/5 = 1).

10 JAMB-Style Examples on Closure, Commutativity, Associativity & Distributivity (with full solutions)

Example 1 (JAMB 1999 style – Closure & Commutativity)
A binary operation * is defined on positive integers by x * y = 2x – 3y + 2. Is the operation (i) closed (ii) commutative on the set of positive integers?
Step 1: For closure, pick x=1, y=2 → 2(1) – 3(2) + 2 = 2 – 6 + 2 = –2
Step 2: –2 is not a positive integer, so result outside the set.
Step 3: Not closed.
Step 4: For commutativity, check x * y = y * x.
Step 5: 1 * 2 = 2(1) – 3(2) + 2 = –2
Step 6: 2 * 1 = 2(2) – 3(1) + 2 = 4 – 3 + 2 = 3
Step 7: –2 ≠ 3, so not commutative.
Answer: (i) Not closed (ii) Not commutative
Example 2 (JAMB style – Closure on reals)
The operation a ∗ b = a + b – ab is defined on the set of real numbers ℝ. Is it closed?
Step 1: Take any two reals a and b, compute a + b – ab.
Step 2: Addition a + b is real.
Step 3: Product ab is real.
Step 4: Subtracting reals is real.
Step 5: So result is always real, inside ℝ.
Step 6: Closed on ℝ.
Answer: Closed on ℝ
Example 3 (JAMB style – Commutativity)
Given a ∗ b = a + b – ab on ℝ. Show it is commutative.
Step 1: Compute a ∗ b = a + b – ab
Step 2: Compute b ∗ a = b + a – ba
Step 3: Since a + b = b + a and ab = ba, b + a – ba = a + b – ab
Step 4: a ∗ b = b ∗ a
Step 5: Commutative.
Answer: Commutative
Example 4 (JAMB style – Associativity)
Let a ∗ b = a + b – ab on ℝ. Check if it is associative.
Step 1: Compute (a ∗ b) ∗ c
Step 2: a ∗ b = a + b – ab
Step 3: (a + b – ab) ∗ c = (a + b – ab) + c – (a + b – ab)c
Step 4: Expand: a + b – ab + c – ac – bc + abc = a + b + c – ab – ac – bc + abc
Step 5: Now a ∗ (b ∗ c)
Step 6: b ∗ c = b + c – bc
Step 7: a ∗ (b + c – bc) = a + (b + c – bc) – a(b + c – bc)
Step 8: Expand: a + b + c – bc – ab – ac + abc = a + b + c – ab – ac – bc + abc
Step 9: Both sides equal.
Step 10: Associative.
Answer: Associative
Example 5 (JAMB style – Distributivity)
Let * be defined by a ∗ b = a + b – ab and ordinary addition +. Does * distribute over +?
Step 1: Compute a ∗ (b + c) = a + (b + c) – a(b + c) = a + b + c – ab – ac
Step 2: Compute (a ∗ b) + (a ∗ c) = (a + b – ab) + (a + c – ac) = 2a + b + c – ab – ac
Step 3: a + b + c – ab – ac ≠ 2a + b + c – ab – ac
Step 4: Not equal.
Step 5: Does not distribute over +.
Answer: Does not distribute over +
Example 6 (JAMB 2008 style)
m * n = n – (m + 2) on real numbers. Is it commutative?
Step 1: Compute 3 * (–5) = –5 – (3 + 2) = –5 – 5 = –10
Step 2: Compute (–5) * 3 = 3 – (–5 + 2) = 3 – (–3) = 3 + 3 = 6
Step 3: –10 ≠ 6
Step 4: Not commutative.
Answer: Not commutative
Example 7 (JAMB style – Associativity check)
If a ∗ b = ab on positive integers, is it associative?
Step 1: Compute (2 ∗ 3) ∗ 4 = (2×3) × 4 = 6 × 4 = 24
Step 2: Compute 2 ∗ (3 ∗ 4) = 2 × (3×4) = 2 × 12 = 24
Step 3: Equal for these.
Step 4: Multiplication is associative in general, so yes.
Answer: Associative
Example 8 (JAMB style – Closure on integers)
x Δ y = x + y + 1 on integers ℤ. Is it closed?
Step 1: Take x = 1, y = –2 → 1 + (–2) + 1 = 0, integer.
Step 2: Take x = –3, y = 4 → –3 + 4 + 1 = 2, integer.
Step 3: Sum of integers + 1 is always integer.
Step 4: Closed.
Answer: Closed
Example 9 (JAMB style – Distributivity)
Suppose ⊕ is ordinary addition and ⊗ is a ∗ b = 2ab. Does ⊗ distribute over ⊕?
Step 1: a ⊗ (b ⊕ c) = 2a(b + c) = 2ab + 2ac
Step 2: (a ⊗ b) ⊕ (a ⊗ c) = 2ab + 2ac
Step 3: Equal.
Step 4: Distributes.
Answer: Distributes
Example 10 (JAMB style – Mixed properties)
The operation a ∗ b = a² – b² on reals.
Is it (i) closed (ii) commutative?
Step 1: For closure, a² – b² = (a – b)(a + b), product of reals is real, so closed.
Step 2: For commutative, 3 ∗ 2 = 9 – 4 = 5
Step 3: 2 ∗ 3 = 4 – 9 = –5
Step 4: 5 ≠ –5, not commutative.
Answer: (i) Closed (ii) Not commutative

10 JAMB-Style Examples on Identity and Inverse Elements (with full solutions)

Example 1 (JAMB 1999)
a * b = ab + a + b, identity element is 0. Find inverse of 2.
Step 1: Let inverse of 2 be x.
Step 2: 2 * x = 0
Step 3: 2x + 2 + x = 0
Step 4: 3x + 2 = 0
Step 5: 3x = –2 → x = –2/3
Answer: –2/3
Example 2 (JAMB 2009)
m ⊗ n = m + n + mn, identity = 0. Find inverse of –5.
Step 1: Let inverse of –5 be x.
Step 2: –5 ⊗ x = 0
Step 3: –5 + x + (–5)x = 0
Step 4: –5 + x – 5x = 0
Step 5: –5 – 4x = 0 → –4x = 5 → x = –5/4
Answer: –5/4
Example 3 (JAMB 2007)
a Δ b = a + b + 1, identity = –1. Find inverse of 7.
Step 1: Let inverse of 7 be x.
Step 2: 7 Δ x = –1
Step 3: 7 + x + 1 = –1
Step 4: x + 8 = –1
Step 5: x = –9
Answer: –9
Example 4 (JAMB 2012 style)
p * q = p + q – pq, identity = 0. Find inverse of p (p ≠ 1).
Step 1: Let inverse of p be x.
Step 2: p * x = 0
Step 3: p + x – px = 0
Step 4: x – px = –p
Step 5: x(1 – p) = –p
Step 6: x = –p / (1 – p) = p / (p – 1)
Answer: p / (p – 1)
Example 5 (JAMB 2021 style)
m * n = mn + m + n, identity e satisfies m * e = m. Find e.
Step 1: m * e = m
Step 2: m e + m + e = m
Step 3: m e + e = 0
Step 4: e (m + 1) = 0
Step 5: Since m can be any, e = 0
Answer: 0
Example 6 (JAMB 2014 style)
x * y = x y (ordinary multiplication). Given x * 2 = 12 – x. Find possible x.
Step 1: x * 2 = x × 2 = 12 – x
Step 2: 2x = 12 – x
Step 3: 2x + x = 12
Step 4: 3x = 12
Step 5: x = 4
Answer: 4
Example 7 (JAMB style)
a * b = (a + b)/2 on reals, identity = 2. Find inverse of 6.
Step 1: Let inverse of 6 be x.
Step 2: 6 * x = 2
Step 3: (6 + x)/2 = 2
Step 4: 6 + x = 4
Step 5: x = 4 – 6 = –2
Answer: –2
Example 8 (JAMB 2020 style)
a x b = a² + b, identity e. Find e.
Step 1: a x e = a
Step 2: a² + e = a
Step 3: e = a – a²
Step 4: e depends on a, so no single e for all a.
Step 5: No identity element.
Answer: No identity
Example 9 (JAMB style)
The operation * on reals defined by a * b = ab/2. Identity element is 2. Find inverse of –4.
Step 1: Let inverse of –4 be x.
Step 2: –4 * x = 2
Step 3: (–4 x)/2 = 2
Step 4: –2x = 2
Step 5: x = –1
Answer: –1
Example 10 (JAMB 1999 variant)
x * y = 2x – 3y + 2 on positive integers. If identity exists, find it.
Step 1: x * e = x
Step 2: 2x – 3e + 2 = x
Step 3: x + 2 – 3e = x
Step 4: 2 – 3e = 0
Step 5: 3e = 2 → e = 2/3 (not integer, so no identity on positive integers)
Answer: No identity
