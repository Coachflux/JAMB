📚 Chapter 9: Binary Operations

🎯 Learning Objectives: Understand binary operations, their properties (closure, commutative, associative, identity, inverse, distributive), and solve problems involving defined operations.

What is a Binary Operation? 🤔

A binary operation is a rule that combines two elements of a set to produce one element from the same set.

Definition:
If we have a set S and an operation *, then for any a, b ∈ S:
a * b = c   where c ∈ S

Common symbols: *, ∘, ⊕, ⊗, Δ, ∇ (or any defined rule)
1️⃣ PROPERTIES OF BINARY OPERATIONS

A. Closure Property 🔒

A set S is closed under operation * if for all a, b ∈ S:

a * b ∈ S

🔢 Example: Closure

Set of even numbers E = {2, 4, 6, 8, ...}, Operation: Addition

Check:
4 + 6 = 10 ∈ E ✓

Therefore: Closed under addition

B. Commutative Property 🔄

a * b = b * a

🔢 Example: Commutative

Operation: a * b = a + b + ab

Check:
2 * 3 = 2 + 3 + (2)(3) = 11
3 * 2 = 3 + 2 + (3)(2) = 11

Therefore: 2 * 3 = 3 * 2 ✓
Commutative

C. Associative Property 🔗

(a * b) * c = a * (b * c)

🔢 Example: Associative

Operation: a * b = a + b - 1

Check:
LHS: (2 * 3) * 4
= (2+3-1) * 4
= 4 * 4 = 4+4-1 = 7

RHS: 2 * (3 * 4)
= 2 * (3+4-1)
= 2 * 6 = 2+6-1 = 7

LHS = RHS ✓ Associative

D. Identity Element 🆔

An element e such that:

a * e = e * a = a

🔢 Example: Finding Identity

Operation: a * b = a + b - 5

Solution:
Find e such that a * e = a
a + e - 5 = a
e = 5

Identity element = 5

E. Inverse Element 🔄

For element a, its inverse a-1 satisfies:

a * a-1 = a-1 * a = e (identity)

🔢 Example: Finding Inverse

Operation: a * b = a + b - 5 (identity e = 5)

Solution:
Find inverse of 3:
3 * 3-1 = 5
3 + 3-1 - 5 = 5
3-1 = 7

Inverse of 3 is 7

F. Distributive Property 📐

Operation ∘ is distributive over * if:

a ∘ (b * c) = (a ∘ b) * (a ∘ c)
2️⃣ WORKED EXAMPLES (JAMB-STYLE)

🔢 Example 1: Defining and Evaluating

The operation * is defined on the set of real numbers by:

a * b = a + b2

(a) Evaluate 4 * 6:
4 * 6 = 4 + 62 = 102 = 5
Answer: 5
(b) Evaluate (2 * 8) * 4:
First: 2 * 8 = 2+82 = 5
Then: 5 * 4 = 5+42 = 4.5
Answer: 4.5

🔢 Example 2: Finding Identity Element

Operation: a * b = a + b + ab. Find the identity element.

Solution:
Let e be the identity element.
a * e = a
a + e + ae = a
e + ae = 0
e(1 + a) = 0

For this to hold for all a: e = 0

Verification: a * 0 = a + 0 + a(0) = a ✓
Answer: Identity = 0

🔢 Example 3: Finding Inverse Element

Using a * b = a + b + ab with identity e = 0, find the inverse of 2.

Solution:
Let 2-1 = x
2 * x = 0
2 + x + 2x = 0
2 + 3x = 0
x = -23
Answer: Inverse of 2 is -2/3

🔢 Example 4: Checking Properties

Operation * on ℝ is defined by a * b = 2a + b - 3

(a) Is it commutative?
• a * b = 2a + b - 3
• b * a = 2b + a - 3

Since 2a + b - 3 ≠ 2b + a - 3
NOT commutative ✗
(b) Is it associative?
• (a * b) * c = (2a + b - 3) * c
= 2(2a + b - 3) + c - 3
= 4a + 2b - 6 + c - 3
= 4a + 2b + c - 9

• a * (b * c) = a * (2b + c - 3)
= 2a + (2b + c - 3) - 3
= 2a + 2b + c - 6

Since 4a + 2b + c - 9 ≠ 2a + 2b + c - 6
NOT associative ✗

🔢 Example 5: Modulo Arithmetic (Common in JAMB)

Operation ⊕ on set {0, 1, 2, 3, 4} is defined as a ⊕ b = remainder when a + b is divided by 5.

Operation Table: ⊕ | 0 1 2 3 4 ---+---------------- 0 | 0 1 2 3 4 1 | 1 2 3 4 0 2 | 2 3 4 0 1 3 | 3 4 0 1 2 4 | 4 0 1 2 3
(b) Find identity element:
From table: a ⊕ 0 = 0 ⊕ a = a
Identity = 0
(c) Find inverse of 3:
Look for x where 3 ⊕ x = 0
From table: 3 ⊕ 2 = 0
Inverse of 3 is 2
📋 SUMMARY TABLE

Properties & Formulas 📊

Property Definition Formula/Condition How to Check
Closure Result stays in set a * b ∈ S Calculate ab, check if in S
Commutative Order doesn't matter a * b = b * a Swap a and b, compare results
Associative Grouping doesn't matter (a * b) * c = a * (b * c) Calculate both sides, compare
Identity (e) Combines to give same element a * e = e * a = a Solve a * e = a for e
Inverse (a-1) Combines to give identity a * a-1 = e Solve a * x = e for x
Distributive One operation over another a ∘ (b * c) = (a ∘ b) * (a ∘ c) Expand both sides, compare

⚡ Quick Reference: Solving Steps

Evaluate a * b: Substitute values into definition

Check Commutative: Calculate ab and ba, compare

Check Associative: Calculate (ab)c and a(bc), compare

Find Identity: Set a * e = a, solve for e

Find Inverse of a: Set a * x = e, solve for x

Construct Table: Calculate all combinations systematically

🎯 JAMB Exam Tips

  1. Always write out the operation definition clearly before starting
  2. Show all working steps — JAMB awards method marks
  3. For identity/inverse: Substitute back to verify your answer
  4. Modulo operations: Remember a mod n gives remainder 0 to n-1
  5. Tables: Be systematic — row first, then column

📝 Practice Question (Try This!)

An operation * is defined on the set of real numbers by a * b = ab4.

(a) Evaluate 6 * 8

(b) Find the identity element

(c) Find the inverse of 2

(d) Is the operation commutative?

Solutions:

(a) 6 * 8 = 6 × 84 = 484 = 12

(b) a * e = a ⇒ ae4 = a ⇒ e = 4

(c) 2 * x = 4 ⇒ 2x4 = 4 ⇒ x = 8

(d) a * b = ab4 = ba4 = b * a ✓ Yes, commutative

Quick Checklist for Binary Operations:

  • Write down the operation definition clearly
  • For evaluation: substitute values carefully
  • For properties: test with specific numbers first
  • For identity: solve a * e = a for e
  • For inverse: solve a * x = e for x
  • Always verify your answers by substitution
  • For modulo: remember remainder is always less than divisor
