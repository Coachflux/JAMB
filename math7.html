📚 Chapter 7: Inequalities

🎯 Learning Objectives: Solve linear and quadratic inequalities analytically and graphically. Interpret graphs of inequalities.

1️⃣ LINEAR INEQUALITIES

Understanding Inequality Symbols 🔣

Symbol Meaning Example
< Less than x < 5 (x is less than 5)
> Greater than x > 3 (x is greater than 3)
≤ Less than or equal to x ≤ 4 (x is 4 or less)
≥ Greater than or equal to x ≥ 2 (x is 2 or more)

⚠️ Key Rule

The inequality sign FLIPS when you multiply or divide both sides by a NEGATIVE number!

Solving Linear Inequalities 📝

🔢 Example 1: Basic Linear Inequality

Solve: 3x + 7 < 16

Solution:
3x + 7 < 16
3x < 16 - 7
3x < 9
x < 3
Answer: x < 3 or (-∞, 3)

🔢 Example 2: When Sign Flips!

Solve: 5 - 2x ≥ 11

Solution:
5 - 2x ≥ 11
-2x ≥ 11 - 5
-2x ≥ 6
x ≤ 6-2   (Sign flips!)
x ≤ -3
Answer: x ≤ -3 or (-∞, -3]

🔢 Example 3: Compound Inequality

Solve: -3 ≤ 2x + 5 < 13

Solution: Solve all parts simultaneously:
-3 ≤ 2x + 5 < 13
-3 - 5 ≤ 2x < 13 - 5
-8 ≤ 2x < 8
-4 ≤ x < 4
Answer: -4 ≤ x < 4 or [-4, 4)

Graphical Representation 📊

Graphing Rules:

• < or > → Use open circle (○)

• ≤ or ≥ → Use closed circle (●)

Inequality Graph Representation
x > 2 Open circle at 2, arrow right →
x ≤ -1 Closed circle at -1, arrow left ←
-2 < x ≤ 3 Open at -2, closed at 3, line between

Linear Inequalities with Fractions 📐

🔢 Example 4: With Fractions

Solve: x3 + x2 > 5

Solution:
Find LCM of 3 and 2 = 6
Multiply every term by 6:

6 × x3 + 6 × x2 > 6 × 5
2x + 3x > 30
5x > 30
x > 6
Answer: x > 6
2️⃣ QUADRATIC INEQUALITIES
💡 JAMB Scope: Only quadratic inequalities that factorize nicely with whole number (integral) roots.

General Approach 🎯

⚡ 5-Step Method

1. Rewrite in standard form (make RHS = 0)

2. Factorize the quadratic expression

3. Find critical values (roots)

4. Sketch the parabola or use test intervals

5. Select appropriate intervals based on inequality sign

🔢 Example 5: Basic Quadratic Inequality

Solve: x² - 5x + 6 > 0

Solution:
x² - 5x + 6 > 0
(x - 2)(x - 3) > 0

Critical values: x = 2 and x = 3

Interval Testing:
• x < 2: Test x = 0 → (-2)(-3) = 6 [+]
• 2 < x < 3: Test x = 2.5 → (0.5)(-0.5) = -0.25 [-]
• x > 3: Test x = 4 → (2)(1) = 2 [+]

We want where > 0 (positive): x < 2 or x > 3
Answer: x < 2 or x > 3, or (-∞, 2) ∪ (3, ∞)

🔢 Example 6: "Less Than" Quadratic

Solve: x² + 3x - 10 ≤ 0

Solution:
x² + 3x - 10 ≤ 0
(x + 5)(x - 2) ≤ 0

Critical values: x = -5 and x = 2

Since parabola opens upward and we want ≤ 0 (negative or zero), we take the middle interval.
Answer: -5 ≤ x ≤ 2 or [-5, 2]

🔢 Example 7: Rearranging First

Solve: 2x² < 5x + 3

Solution:
2x² - 5x - 3 < 0
(2x + 1)(x - 3) < 0

Critical values: x = -12 and x = 3

Test intervals for < 0 (negative): -12 < x < 3
Answer: -½ < x < 3 or (-½, 3)

🔢 Example 8: Perfect Square (Special Case)

Solve: x² - 6x + 9 > 0

Solution:
x² - 6x + 9 > 0
(x - 3)² > 0

Since (x-3)² is always ≥ 0, and equals 0 only at x = 3:

All real numbers except x = 3
Answer: x ≠ 3 or (-∞, 3) ∪ (3, ∞)
3️⃣ GRAPHICAL SOLUTIONS

Graphing Linear Inequalities 📈

Inequality Region to Shade
y > mx + c Above the line
y < mx + c Below the line
y ≥ mx + c Above the line (solid line)
y ≤ mx + c Below the line (solid line)

🔢 Example 9: Graphing Inequality

Graph the inequality: 2x + 3y ≤ 6

Steps:
1. Draw the line 2x + 3y = 6 (solid line because of ≤)
• When x = 0, y = 2 → point (0, 2)
• When y = 0, x = 3 → point (3, 0)
2. Shade below the line (since it's "less than")
3. Test point (0,0): 0 ≤ 6 ✓ → Origin side is the solution

🔢 Example 10: Simultaneous Inequalities

Find the region satisfying:

• x + y ≤ 4

• x ≥ 0

• y ≥ 0

Solution:
This defines a feasible region in the first quadrant below the line x + y = 4.

Vertices of the region: (0,0), (4,0), and (0,4)
📋 SUMMARY TABLE

Key Formulas & Rules 📊

Category Formula/Rule Notes
Sign Change If a > b, then -a < -b Multiplying/dividing by negative flips sign
Transitive If a > b and b > c, then a > c Logical ordering
Addition If a > b, then a + c > b + c Adding same number preserves inequality
Multiplication (+) If a > b and c > 0, then ac > bc Positive multiplication preserves sign
Multiplication (-) If a > b and c < 0, then ac < bc Negative multiplication reverses sign
Quadratic (>) For (x-a)(x-b) > 0 where a < b Solution: x < a or x > b (outer intervals)
Quadratic (<) For (x-a)(x-b) < 0 where a < b Solution: a < x < b (inner interval)
Number Line x > a Open circle at a, arrow right
Number Line x ≤ b Closed circle at b, arrow left
Graphical y > mx + c Shade above the line
Graphical y < mx + c Shade below the line

⚠️ Common Mistakes to Avoid

Mistake Correction
Forgetting to flip sign when multiplying/dividing by negative Always flip when multiplying/dividing by negative
Including endpoints for strict inequalities (< or >) Use open circles and parentheses ()
Wrong interval for quadratic inequalities Sketch the parabola or test points
Dividing by variable without checking if positive/negative Bring all terms to one side instead

🎓 Exam Tips

  1. Always check if your answer makes sense by testing a value
  2. Sketch number lines for quadratic inequalities to avoid confusion
  3. Watch for "integral roots only" — JAMB quadratics will factorize nicely
  4. Remember the sign flip rule — it's tested frequently!
  5. For graphical questions, identify the boundary line first, then determine the shading region

📝 Practice Questions (JAMB Style)

Q1. Solve: 3(x - 2) ≤ 2(x + 1) - 5

Q2. Solve: x² - 7x + 10 < 0

Q3. Find the range of values of x for which x+12 - x-13 > 1

Quick Checklist for Inequality Problems:

  • Identify if linear or quadratic inequality
  • Remember: flip sign when × or ÷ by negative
  • For quadratics: factorize and find critical values
  • Use test points or sketch parabola for quadratics
  • For > 0: take outer intervals; for < 0: take inner interval
  • Check if endpoints should be included (≤, ≥) or excluded (<, >)
  • For fractions: find LCM and clear denominators
