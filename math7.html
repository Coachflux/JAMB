Linear Inequalities (Analytical and Graphical Solutions)

Abumchukwu, linear inequalities are solved almost like equations, but flip the inequality sign when multiplying or dividing by a negative number. The goal is to isolate x (or the variable). I will teach each part very slowly and simply, like we are in class together. Read one example at a time. Try it on paper first, then check my steps. You will understand everything fast.

Analytical Solution Steps (1 variable)

Step 1: Simplify both sides (expand brackets, collect like terms).
Step 2: Move variable terms to one side, constants to the other.
Step 3: Divide or multiply — flip sign if the coefficient is negative.
For compound inequalities (e.g., –3 ≤ 2x + 1 < 5), solve each part separately and find the overlapping range.

Graphical Solution (1 variable – Number Line)

Step 1: > or < → open circle (○), arrow in the direction.
Step 2: ≥ or ≤ → closed circle (●), arrow in the direction.
Step 3: Compound: shade only the overlapping part.

Graphical Solution (2 variables – Shaded Region on Plane)

Step 1: Graph each boundary line (solid for ≥/≤, dashed for >/ <).
Step 2: Test a point (usually (0,0)) in the inequality.
Step 3: Shade the side that satisfies it.
Step 4: The solution is the overlapping shaded region (often a polygon like a triangle).
Candidates must interpret graphs: identify which system of inequalities produces a given shaded region (common JAMB question).

10 JAMB-Style Examples for Linear Inequalities (with full solutions)

Example 1: Solve 3(x – 1) ≤ 2(x – 3)
Step 1: Expand brackets: 3x – 3 ≤ 2x – 6
Step 2: Subtract 2x from both sides: x – 3 ≤ –6
Step 3: Add 3 to both sides: x ≤ –3
Answer: x ≤ –3
Example 2: For what range of values of x is ½x + ¼ > ⅓x + ½?
Step 1: Multiply by 12 to clear fractions: 6x + 3 > 4x + 6
Step 2: 6x – 4x > 6 – 3 → 2x > 3
Step 3: x > 3/2
Answer: x > 3/2
Example 3: Find the range of values of x which satisfy 4x – 7 ≤ 3x and 3x – 4 ≤ 4x.
Step 1: First: 4x – 7 ≤ 3x → x ≤ 7
Step 2: Second: 3x – 4 ≤ 4x → –4 ≤ x
Step 3: Combine: –4 ≤ x ≤ 7
Answer: –4 ≤ x ≤ 7
Example 4: Find the range of values of x for which 3x – 7 ≤ 0 and x + 5 > 0.
Step 1: 3x – 7 ≤ 0 → x ≤ 7/3
Step 2: x + 5 > 0 → x > –5
Step 3: Combine: –5 < x ≤ 7/3
Answer: –5 < x ≤ 7/3
Example 5: Solve the inequalities –6 ≤ 4 – 2x < 5 – x.
Step 1: Split: –6 ≤ 4 – 2x and 4 – 2x < 5 – x
Step 2: –6 ≤ 4 – 2x → –10 ≤ –2x → 5 ≥ x (flip) → x ≤ 5
Step 3: 4 – 2x < 5 – x → –2x + x < 5 – 4 → –x < 1 → x > –1 (flip)
Step 4: Combine: –1 < x ≤ 5
Answer: –1 < x ≤ 5
Example 6: List all integers satisfying –2 < 2x – 6 < 4.
Step 1: Add 6: 4 < 2x < 10
Step 2: Divide by 2: 2 < x < 5
Step 3: Integers: 3, 4
Answer: 3, 4
Example 7: Triangle SPT is the solution of the linear inequalities (graph shows triangle in second quadrant).
Step 1: Identify lines: 2y – x – 2 = 0, y + 2x + 2 = 0, y = 0, x = 0
Step 2: Test point in shaded region to confirm inequalities (≤ or <)
Step 3: 2y – x – 2 ≤ 0, y + 2x + 2 ≤ 0, y ≥ 0, x ≤ 0
Answer: 2y – x – 2 ≤ 0, y + 2x + 2 ≤ 0, y ≥ 0, x ≤ 0
Example 8: Which inequality describes the graph (shaded region below the line 4y + 5x = 20 in first quadrant)?
Step 1: Line: 4y + 5x = 20
Step 2: Test (0,0): 0 + 0 = 0 < 20
Step 3: Below line shaded, solid line → 4y + 5x ≤ 20
Answer: 4y + 5x ≤ 20
Example 9: The graph above represents inequalities (shaded triangle in first quadrant bounded by 3x + 4y = 12).
Step 1: Line: 3x + 4y = 12
Step 2: Test (0,0): 0 + 0 = 0 ≤ 12 true
Step 3: First quadrant, solid line → 3x + 4y ≤ 12, x ≥ 0, y ≥ 0

Example 10: Solve x² – 4x + 3 > 0 and x > 1.
Step 1: (x – 1)(x – 3) > 0 → x < 1 or x > 3

(b) Quadratic Inequalities with Integral Roots Only

Very easy explanation

Bring to standard form ax² + bx + c ≥ 0 (or >, ≤, <).
Factorise: (x – r)(x – s) where r and s are integers.
Roots divide number line into intervals.
Use sign chart or test points in each interval.
Include roots if ≥ or ≤.

Sign Chart Rule: Product positive when both factors same sign (outside roots or between if negative leading coefficient).

Graphical Interpretation: The quadratic graph is a parabola; solution is where it is above/below x-axis (including touches for ≥/≤).

10 JAMB-Style Examples for Quadratic Inequalities (with full solutions)
