🔑 ESSENTIAL FORMULAS

Summary Table 📊

Application Formula/Rule Explanation
Rate of Change dy/dx or ds/dt Instantaneous rate of change
Chain Rule dy/dt = dy/dx × dx/dt Connects related rates
Stationary Points Set dy/dx = 0 Points where gradient is zero
Second Derivative d²y/dx² Determines nature of points
Maximum Point dy/dx = 0 & d²y/dx² < 0 Curve concave downward (peak)

1️⃣ RATE OF CHANGE

Concept & Applications 📈

Key Rates:
• ds/dt = rate of change of distance (speed/velocity)
• dA/dt = rate of change of area
• dV/dt = rate of change of volume

🔢 Example 1: Basic Rate of Change

Square side increasing at 2 cm/s. Find rate of increase of area when side is 5 cm.

Let s = side, A = area
Given: ds/dt = 2 cm/s

Formula: A = s²

Differentiate with respect to t:
dA/dt = 2s · ds/dt (chain rule)

Substitute s = 5 and ds/dt = 2:
dA/dt = 2(5)(2) = 20 cm²/s

🔢 Example 2: Related Rates (JAMB Style)

Spherical balloon inflated, radius increases at 0.5 cm/s. Find rate of volume increase when r = 3 cm. [π = 227]

Volume: V = 43πr³
Given: dr/dt = 0.5 cm/s

Differentiate V with respect to t:
dV/dt = 43π · 3r² · dr/dt = 4πr² · dr/dt

When r = 3:
dV/dt = 4 × 227 × 9 × 0.5
= 3967 ≈ 56.57 cm³/s
2️⃣ MAXIMA AND MINIMA

Optimization Problems 📊

💡 Step-by-Step Procedure:
1. Find dy/dx
2. Set to zero: Solve dy/dx = 0
3. Second derivative test: Calculate d²y/dx²
4. Determine nature: >0 (min), <0 (max), =0 (inflection)

🔢 Example 3: Finding Max/Min Points

Find maximum and minimum values of y = x³ - 6x² + 9x + 2

Step 1: Differentiate
dy/dx = 3x² - 12x + 9

Step 2: Set to zero
3x² - 12x + 9 = 0
x² - 4x + 3 = 0
(x - 1)(x - 3) = 0
x = 1 or x = 3

Step 3: Second derivative
d²y/dx² = 6x - 12

Step 4: Test each point
At x = 1: 6(1) - 12 = -6 < 0 → Maximum
y = 1 - 6 + 9 + 2 = 6 → Point (1, 6)

At x = 3: 6(3) - 12 = 6 > 0 → Minimum
y = 27 - 54 + 27 + 2 = 2 → Point (3, 2)

🔢 Example 4: Optimization (JAMB Common Type)

Farmer has 100 m fencing for rectangular plot. Find dimensions for maximum area.

Let length = x, width = y

Constraint (perimeter):
2x + 2y = 100 → x + y = 50 → y = 50 - x

Area: A = x · y = x(50 - x) = 50x - x²

Differentiate: dA/dx = 50 - 2x

Set to zero: 50 - 2x = 0 → x = 25

Second derivative: d²A/dx² = -2 < 0 ✓ (Maximum)

Find y: y = 50 - 25 = 25

Maximum Area = 25 × 25 = 625 m²
Square plot: 25 m × 25 m, Area = 625 m²

🔢 Example 5: Curve Sketching

Find coordinates of turning point on y = 2x² - 8x + 5 and determine its nature.

dy/dx = 4x - 8

Set to zero: 4x - 8 = 0 → x = 2

Find y: y = 2(2)² - 8(2) + 5 = 8 - 16 + 5 = -3

Second derivative: d²y/dx² = 4 > 0
Turning point at (2, -3), Minimum
📝 JAMB PRACTICE QUESTIONS

Objective Questions 🎯

Q1: If y = x³ - 3x² + 4, find x at which y is maximum.

(A) 0 (B) 1 (C) 2 (D) 3

dy/dx = 3x² - 6x = 0
3x(x - 2) = 0 → x = 0 or 2

d²y/dx² = 6x - 6
At x=0: -6 < 0 (max) ✓
At x=2: 6 > 0 (min)

Answer: (A) 0

Q2: Circle radius increasing at 0.1 cm/s. Rate of area increase when r = 5 cm?

(A) π (B) 2π (C) 5π (D) 10π cm²/s

A = πr²
dA/dt = 2πr · dr/dt
= 2π(5)(0.1) = π

Answer: (A) π cm²/s

🎯 Key Points for JAMB 2026

✓ Simplify first: Factorize before differentiating

✓ Chain rule crucial: Essential for related rates

✓ Don't forget units: Rate questions require units

✓ Test both points: Some curves have max AND min

✓ Second derivative saves time: Quicker than testing values

✓ Read carefully: "Rate of increase" vs "rate of decrease"

⚡ Quick Reference: Nature of Points

d²y/dx² > 0 → Minimum (valley) 📉

d²y/dx² < 0 → Maximum (peak) 📈

d²y/dx² = 0 → Point of inflection
