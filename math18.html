📚 Introduction to Integration

Integration is the reverse process of differentiation. If differentiation gives us the rate of change, integration helps us find the original function from its rate of change.

Key Terms:
• Indefinite Integration: General function (+C)
• Definite Integration: Numerical value between limits
• Constant of Integration (+C): Derivative of any constant is zero
💡 JAMB Focus: Expect 3-5 questions on integration, combining algebraic, trigonometric, and application problems.
1️⃣ BASIC INTEGRATION RULES

Power Rule ⭐ (Most Important!)

∫ xⁿ dx = xⁿ⁺¹n+1 + C (where n ≠ -1)

Constant Multiple: ∫ k·f(x) dx = k ∫ f(x) dx
Sum/Difference: ∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx

🔢 Example 1: Power Rule

Find ∫ 3x⁴ dx

∫ 3x⁴ dx = 3 · x⁴⁺¹4+1 + C
= 3 · x⁵5 + C = 3x⁵5 + C

🔢 Example 2: Polynomial

Find ∫ (2x³ - 5x² + 4x - 7) dx

= 2·x⁴4 - 5·x³3 + 4·x²2 - 7x + C
= x⁴2 - 5x³3 + 2x² - 7x + C
2️⃣ ALGEBRAIC FUNCTIONS

Rational & Exponential Functions 📊

When n = -1: ∫ 1x dx = ln|x| + C

Exponential:
∫ eˣ dx = eˣ + C
∫ aˣ dx = aˣln a + C

🔢 Example 3: Rational Function

Find ∫ 3x dx

∫ 3x dx = 3 ln|x| + C
3️⃣ TRIGONOMETRIC FUNCTIONS

Basic Trigonometric Integrals 📐

Function Integral
sin x -cos x + C
cos x sin x + C
sec² x tan x + C
csc² x -cot x + C
sec x tan x sec x + C
csc x cot x -csc x + C

🔢 Example 4: Trigonometric

Find ∫ (4 sin x + 3 cos x) dx

= 4(-cos x) + 3(sin x) + C
= -4 cos x + 3 sin x + C

🔢 Example 5: Using Identities

Find ∫ sin² x dx

Use identity: sin² x = 1 - cos 2x2

∫ sin² x dx = ∫ 1 - cos 2x2 dx
= 12∫ 1 dx - 12∫ cos 2x dx
= x2 - sin 2x4 + C
4️⃣ INTEGRATION BY SUBSTITUTION

When to Use 🔄

💡 Use when you have a composite function or a function and its derivative present.

Method:

1. Let u = inner function

2. Find du/dx and rearrange for dx

3. Substitute into the integral

4. Integrate with respect to u

5. Substitute back to x

🔢 Example 6: Substitution

Find ∫ 2x(x² + 3)⁴ dx

Let u = x² + 3, then du/dx = 2x, so du = 2x dx

∫ 2x(x² + 3)⁴ dx = ∫ u⁴ du
= u⁵5 + C
= (x² + 3)⁵5 + C

🔢 Example 7: Trigonometric Substitution

Find ∫ cos 3x dx

Let u = 3x, then du = 3 dx, so dx = du3

∫ cos 3x dx = ∫ cos u · du3
= 13 sin u + C
= sin 3x3 + C
5️⃣ INTEGRATION BY PARTS

Formula & LIATE Rule 📋

∫ u dv = uv - ∫ v du

LIATE Rule (Priority for choosing u):
1. Logarithmic | 2. Inverse trig | 3. Algebraic
4. Trigonometric | 5. Exponential

🔢 Example 8: Integration by Parts

Find ∫ x eˣ dx

Let u = x (Algebraic), dv = eˣ dx
Then du = dx, v = eˣ

∫ x eˣ dx = x·eˣ - ∫ eˣ dx
= xeˣ - eˣ + C
= eˣ(x - 1) + C

🔢 Example 9: Another Parts Example

Find ∫ x ln x dx

Let u = ln x (Logarithmic), dv = x dx
Then du = 1xdx, v = x²2

∫ x ln x dx = x²2 ln x - ∫ x²2 · 1x dx
= x²2 ln x - ∫ x2 dx
= x²2 ln x - x²4 + C
= x²4(2 ln x - 1) + C
6️⃣ DEFINITE INTEGRATION

Format & Properties 📊

∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) - F(a)

Properties:
• ∫ₐᵇ f(x) dx = -∫ᵇₐ f(x) dx
• ∫ₐₐ f(x) dx = 0
• ∫ₐᵇ f(x) dx + ∫ᵇᶜ f(x) dx = ∫ₐᶜ f(x) dx

🔢 Example 10: Definite Integral

Evaluate ∫₁³ (2x + 1) dx

∫₁³ (2x + 1) dx = [x² + x]₁³
= (3² + 3) - (1² + 1)
= (9 + 3) - (1 + 1)
= 12 - 2 = 10

🔢 Example 11: Definite with Substitution

Evaluate ∫₀² x(x² + 1)³ dx

Let u = x² + 1, du = 2x dx, so x dx = du2

When x = 0, u = 1; when x = 2, u = 5

∫₀² x(x² + 1)³ dx = ∫₁⁵ u³ · du2
= 12[u⁴4]₁⁵
= 18(5⁴ - 1⁴)
= 18(625 - 1) = 6248 = 78
7️⃣ APPLICATIONS OF INTEGRATION

A. Area Under a Curve 📐

Area = ∫ₐᵇ y dx (above x-axis)

🔢 Example 12: Area

Find area bounded by y = x², x = 1, x = 3, and x-axis.

Area = ∫₁³ x² dx = [x³3]₁³
= 273 - 13 = 263 square units

B. Area Between Two Curves 📊

Area = ∫ₐᵇ (yupper - ylower) dx

🔢 Example 13: Area Between Curves

Find area enclosed by y = x² and y = x.

Find intersection: x² = x → x = 0, 1

Area = ∫₀¹ (x - x²) dx
= [x²2 - x³3]₀¹
= (12 - 13) - 0 = 16 square units

C. Volume of Revolution 🔄

About x-axis: V = π ∫ₐᵇ y² dx

🔢 Example 14: Volume

Find volume generated by rotating y = √x from x = 0 to x = 4 about x-axis.

V = π ∫₀⁴ (√x)² dx = π ∫₀⁴ x dx
= π[x²2]₀⁴
= π(162 - 0) = 8π cubic units

Substitution: ∫ f(g(x))g'(x)dx = ∫ f(u)du

Parts: ∫ u dv = uv - ∫ v du
