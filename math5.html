1️⃣ UNDERSTANDING POLYNOMIALS

What is a Polynomial? 🤔

A polynomial is an expression consisting of variables and coefficients, involving only addition, subtraction, multiplication, and non-negative integer exponents.

General Form:
P(x) = anxn + an-1xn-1 + an-2xn-2 + ... + a1x + a0

Where:
• an, an-1, ... are coefficients (constants)
• x is the variable
• n is the degree (highest power of x)
• a0 is the constant term

Types by Degree (JAMB Focus) 📊

Degree Name General Form Example
0 Constant P(x) = a P(x) = 5
1 Linear P(x) = ax + b P(x) = 2x + 3
2 Quadratic P(x) = ax² + bx + c P(x) = x² - 4x + 4
3 Cubic P(x) = ax³ + bx² + cx + d P(x) = 2x³ - x² + 3x - 1
💡 JAMB Note: You only need to handle polynomials up to degree 3.
2️⃣ CHANGE OF SUBJECT OF FORMULA

Making a Variable the Subject 📝

Rearranging the formula to isolate a specific variable.

🔢 Example 1: Linear Formula

Make x the subject: y = 3x + 7

Solution:
y = 3x + 7
y - 7 = 3x
x = y - 73

🔢 Example 2: Quadratic Formula

Make x the subject: y = 2x² - 8

Solution:
y = 2x² - 8
y + 8 = 2x²
x² = y + 82
x = ±√(y + 82)

🔢 Example 3: With Root (JAMB Style)

Make x the subject: T = 2π√(xg)

Solution:
T2π = √(xg)
(T2π)² = xg
x = g(T2π)² = gT²4π²
3️⃣ MULTIPLICATION & DIVISION

Multiplication ✖️

🔢 Example 4: Multiply (2x + 3)(x² - 2x + 4)

Solution:
= 2x(x² - 2x + 4) + 3(x² - 2x + 4)
= 2x³ - 4x² + 8x + 3x² - 6x + 12
= 2x³ - x² + 2x + 12

Division (Long Division) ➗

🔢 Example 5: Divide 2x³ - 3x² + 4x - 5 by x - 2

Solution: 2x² + x + 6 ________________ x - 2 | 2x³ - 3x² + 4x - 5 2x³ - 4x² _________ x² + 4x x² - 2x _______ 6x - 5 6x - 12 _______ 7 ← Remainder
Answer: 2x² + x + 6 + 7x-2
4️⃣ FACTOR & REMAINDER THEOREMS

The Remainder Theorem 📐

If a polynomial P(x) is divided by (x - a), the remainder is P(a).

The Factor Theorem ✅

If P(a) = 0, then (x - a) is a factor of P(x).

🔢 Example 6: Remainder Theorem

Find the remainder when P(x) = 2x³ - 5x² + 3x - 7 is divided by (x - 2)

Solution:
P(2) = 2(2)³ - 5(2)² + 3(2) - 7
= 2(8) - 5(4) + 6 - 7
= 16 - 20 + 6 - 7
= -5
Remainder = -5

🔢 Example 7: Factor Theorem

Show that (x - 3) is a factor of P(x) = x³ - 6x² + 11x - 6

Solution:
P(3) = (3)³ - 6(3)² + 11(3) - 6
= 27 - 54 + 33 - 6
= 0

Since P(3) = 0, (x - 3) is a factor. ✓
5️⃣ FACTORIZATION TECHNIQUES

A. Common Factor 🔢

🔢 Example 8: Factorize 6x³ + 9x² - 3x

Solution:
= 3x(2x² + 3x - 1)

B. Difference of Two Squares 📐

a² - b² = (a+b)(a-b)

🔢 Example 9: Factorize 9x² - 16

Solution:
= (3x)² - (4)²
= (3x + 4)(3x - 4)

C. Perfect Square Trinomials ⬜

a² + 2ab + b² = (a+b)²
a² - 2ab + b² = (a-b)²

🔢 Example 10: Factorize 4x² - 12x + 9

Solution:
= (2x)² - 2(2x)(3) + (3)²
= (2x - 3)²

D. Quadratic Trinomials (ax² + bx + c) 📊

🔢 Example 11: Factorize 2x² + 7x + 3

Solution:
Find two numbers that multiply to 2 × 3 = 6 and add to 7 → 6 and 1

= 2x² + 6x + x + 3
= 2x(x + 3) + 1(x + 3)
= (2x + 1)(x + 3)

E. Sum & Difference of Cubes 🧊

a³ + b³ = (a+b)(a² - ab + b²)
a³ - b³ = (a-b)(a² + ab + b²)

🔢 Example 12: Factorize 8x³ - 27

Solution:
= (2x)³ - (3)³
= (2x - 3)(4x² + 6x + 9)

F. Factorization by Grouping 🔗

🔢 Example 13: Factorize x³ + 2x² - x - 2

Solution:
= x²(x + 2) - 1(x + 2)
= (x² - 1)(x + 2)
= (x+1)(x-1)(x+2)
6️⃣ ROOTS OF POLYNOMIALS

Finding Roots 🔍

Roots (or zeros) are values of x that make P(x) = 0.

🔢 Example 14: Find the roots of x³ - 2x² - 5x + 6 = 0

Solution:
Try x = 1: 1 - 2 - 5 + 6 = 0 ✓

So (x - 1) is a factor. Divide to get:
x³ - 2x² - 5x + 6 = (x-1)(x² - x - 6)
= (x-1)(x-3)(x+2) = 0
Roots: x = 1, 3, -2

Relationship Between Roots and Coefficients ⚖️

For ax² + bx + c = 0 with roots α and β:

• Sum of roots: α + β = -ba

• Product of roots: αβ = ca

For ax³ + bx² + cx + d = 0 with roots α, β, γ:

• α + β + γ = -ba

• αβ + αγ + βγ = ca

• αβγ = -da

🔢 Example 15: Using Root Relationships

If α and β are roots of 2x² - 6x + 3 = 0, find α² + β²

Solution:
α + β = 62 = 3
αβ = 32

α² + β² = (α + β)² - 2αβ
= 9 - 2(32)
= 9 - 3 = 6
Answer: 6
7️⃣ SIMULTANEOUS EQUATIONS

One Linear, One Quadratic 📐

⚡ Method: Substitution

Substitute the linear equation into the quadratic equation.

🔢 Example 16: Solve Simultaneous Equations

y = x + 1 ... (1)

x² + y² = 13 ... (2)

Solution:
Substitute (1) into (2):
x² + (x+1)² = 13
x² + x² + 2x + 1 = 13
2x² + 2x - 12 = 0
x² + x - 6 = 0
(x+3)(x-2) = 0

When x = -3: y = -3 + 1 = -2
When x = 2: y = 2 + 1 = 3
Solutions: (-3, -2) and (2, 3)
8️⃣ GRAPHS OF POLYNOMIALS

Key Features to Identify 📊

Feature Description
x-intercepts Where y = 0 (roots)
y-intercept Where x = 0 (constant term)
Turning points Maximum and minimum points
Shape ↑↑ (positive cubic) or ↓↓ (negative cubic)

🔢 Example 17: Sketch y = x² - 4x + 3

Solution:
• Roots: (x-3)(x-1) = 0 → x = 1, 3
• y-intercept: y = 3
• Turning point: x = -b2a = 42 = 2
• Minimum value: y = (2)² - 4(2) + 3 = -1

Vertex: (2, -1)

Finding Maxima & Minima 📈📉

🔢 Example 18: Find Turning Points

For y = x³ - 3x²:

Solution:
dydx = 3x² - 6x = 0
3x(x-2) = 0
x = 0 or 2

• At x = 0: Maximum (y = 0)
• At x = 2: Minimum (y = -4)
📋 ESSENTIAL FORMULAS

Quick Reference Table 📊

Topic Formula When to Use
Remainder Theorem Remainder = P(a) when dividing by (x-a) Finding remainder quickly
Factor Theorem If P(a) = 0, then (x-a) is a factor Testing for factors
Difference of Squares a² - b² = (a+b)(a-b) Two perfect square terms
Perfect Square a² ± 2ab + b² = (a ± b)² Trinomial with 2ab middle term
Sum of Cubes a³ + b³ = (a+b)(a²-ab+b²) Two cubic terms added
Difference of Cubes a³ - b³ = (a-b)(a²+ab+b²) Two cubic terms subtracted
Quadratic Roots x = -b ± √(b²-4ac)2a Solving ax²+bx+c=0
Sum of Roots (Quad) α + β = -ba Without solving equation
Product of Roots (Quad) αβ = ca Without solving equation
Vertex x-coordinate x = -b2a Finding turning points

⚡ Quick JAMB Tips

  1. Always check your factors by expanding them back
  2. Use Remainder Theorem instead of long division for remainders
  3. For cubic factorization: Try factors of constant term ÷ factors of leading coefficient
  4. Sketching graphs: Mark all intercepts first, then turning points
  5. Simultaneous equations: Always substitute the linear into the quadratic
📝 PRACTICE QUESTIONS

Q1: Find the remainder when 3x³ - 2x² + 4x - 5 is divided by (x-2)

Answer: 19

Q2: Factorize completely: x³ - 3x² - 4x + 12

Answer: (x-3)(x-2)(x+2)

Q3: Solve simultaneously: y = 2x - 1 and x² + y² = 5

Answer: (0,-1) and (85, 115)

Q4: If (x-2) is a factor of x³ + ax² - 5x + 2, find a

Answer: a = 1

Q5: Find the minimum value of 2x² - 8x + 5

Answer: -3

Quick Checklist for Polynomial Problems:

  • Check degree of polynomial first
  • Use Remainder Theorem for quick remainder finding
  • Test factors using Factor Theorem (P(a) = 0)
  • For quadratics: check discriminant (b² - 4ac)
  • Remember sum and product of roots formulas
  • For graphs: find x-intercepts, y-intercept, and vertex
  • In simultaneous equations, substitute linear into quadratic
