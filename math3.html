🎯 Learning Objectives: Master laws of indices, logarithmic equations, and surd operations for JAMB success. This chapter appears frequently in exams!

1️⃣ INDICES (Powers/Exponents)

Basic Laws of Indices 📈

Law Formula Example
Multiplication am × an = am+n 2³ × 2⁴ = 2⁷ = 128
Division am ÷ an = am-n 5⁶ ÷ 5² = 5⁴ = 625
Power of Power (am)n = amn (3²)³ = 3⁶ = 729
Zero Index a⁰ = 1 (a ≠ 0) 7⁰ = 1, (-5)⁰ = 1
Negative Index a-n = 1an 2-3 = 18
Fractional Index a1n = √[n]a 813 = 2
General Fractional amn = (√[n]a)m 1634 = 2³ = 8

🔢 Example 1: Simplifying Expressions

Simplify: 3⁵ × 3-23³

Solution:
• Numerator: 3⁵ × 3-2 = 35+(-2) = 3³
• Division: 3³3³ = 33-3 = 3⁰ = 1
Answer: 1

🔢 Example 2: Solving Exponential Equations

Solve: 22x+1 = 8x-1

Solution:
Express both sides with base 2:
• 8 = 2³, so 8x-1 = (2³)x-1 = 23x-3

Therefore: 22x+1 = 23x-3
Equate exponents: 2x + 1 = 3x - 3
1 + 3 = 3x - 2x
x = 4
Answer: x = 4

🔢 Example 3: Fractional Indices

Evaluate: 2723 + 1634

Solution:
• 2723 = (³√27)² = 3² = 9
• 1634 = (⁴√16)³ = 2³ = 8

Answer: 9 + 8 = 17
Answer: 17
2️⃣ LOGARITHMS

Definition 📖

If ax = N, then loga N = x

Where:
• a = base (must be positive and ≠ 1)
• N = number (must be positive)
• x = logarithm

Laws of Logarithms 📊

Law Formula Example
Product Rule loga(MN) = loga M + loga N log₂(8×4) = 3+2 = 5
Quotient Rule loga(MN) = loga M - loga N log₃(279) = 3-2 = 1
Power Rule loga(Mn) = n·loga M log₅(25²) = 2×2 = 4
Change of Base loga b = logc blogc a log₂ 8 = log 8log 2
Log of Base loga a = 1 log₅ 5 = 1
Log of 1 loga 1 = 0 log₇ 1 = 0

Common and Natural Logarithms

• Common logarithms: Base 10 (log₁₀ or simply log)

• Natural logarithms: Base e (ln or logₑ) where e ≈ 2.718

🔢 Example 4: Using Logarithm Laws

Simplify: log₃ 54 + log₃ 2 - log₃ 4

Solution:
• log₃ 54 + log₃ 2 = log₃(54 × 2) = log₃ 108
• log₃ 108 - log₃ 4 = log₃(1084) = log₃ 27
• Since 27 = 3³: log₃ 27 = 3
Answer: 3

🔢 Example 5: Solving Logarithmic Equations

Solve: log₂(x+3) + log₂(x-1) = 3

Solution:
• Combine: log₂[(x+3)(x-1)] = 3
• Convert to exponential: (x+3)(x-1) = 2³ = 8
• Expand: x² + 2x - 3 = 8
• x² + 2x - 11 = 0

Using quadratic formula:
x = -2 ± √(4 + 44)2 = -2 ± √482 = -1 ± 2√3

Since x > 1 (for log₂(x-1) to exist):
x = -1 + 2√3 ≈ 2.46
Answer: x = -1 + 2√3 ≈ 2.46

🔢 Example 6: Change of Base

Evaluate: log₄ 8

Solution:
log₄ 8 = log₂ 8log₂ 4 = 32 = 1.5
Answer: 1.5
3️⃣ SURDS (Irrational Roots)

Definition √

Surds are irrational numbers expressed as roots that cannot be simplified to exact rational numbers.

Examples:

• √2, √3, ³√5

Types of Surds 📊

Type Description Example
Quadratic Surd Square root of non-perfect square √5, √18
Cubic Surd Cube root of non-perfect cube ³√2, ³√7
Binomial Surd Sum/difference of two surds 3 + √2, √5 - √3
Conjugate Surd Change sign between terms If a + √b, conjugate is a - √b

Operations with Surds

Simplification 📝

√(a × b) = √a × √b
√(a² × b) = a√b
💡 Example: √50 = √(25 × 2) = 5√2

Rationalization 🎯

Removing surds from the denominator.

Form Multiply by Result
1√a √a√a √aa
1a + √b a - √ba - √b a - √ba² - b
1√a + √b √a - √b√a - √b √a - √ba - b

🔢 Example 7: Simplifying Surds

Simplify: √72 + √50 - √32

Solution:
• √72 = √(36 × 2) = 6√2
• √50 = √(25 × 2) = 5√2
• √32 = √(16 × 2) = 4√2

Answer: 6√2 + 5√2 - 4√2 = 7√2
Answer: 7√2

🔢 Example 8: Rationalization

Rationalize: 32 + √5

Solution:
Multiply by conjugate (2 - √5):

32 + √5 × 2 - √52 - √5 = 3(2 - √5)4 - 5

= 6 - 3√5-1 = 3√5 - 6
Answer: 3√5 - 6

🔢 Example 9: Binomial Surds

If a = 3 + 2√2 and b = 3 - 2√2, find a² + b²

Solution:
• Note that a + b = 6 and ab = 9 - 8 = 1
• a² + b² = (a+b)² - 2ab = 36 - 2 = 34
Answer: 34

🎯 JAMB Exam Tips

  1. Indices: Always try to express numbers with the same base when solving equations
  2. Logarithms: Remember that loga N is only defined when N > 0
  3. Surds: When rationalizing, the conjugate always has the opposite sign
  4. Common mistake: loga(M + N) ≠ loga M + loga N (product rule only, not sum!)

⚠️ Common Pitfall

Remember: There is no law for loga(M + N)! Many students mistakenly think loga(M + N) = loga M + loga N. This is FALSE!

  • Negative indices mean reciprocals
  • Fractional indices: denominator = root, numerator = power
