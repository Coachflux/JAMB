📚 Chapter 8: Progression (AP & GP)

🎯 Learning Objectives: Understand and apply formulas for Arithmetic and Geometric Progressions to solve problems involving nth terms, sums, and means.

1️⃣ ARITHMETIC PROGRESSION (AP)

What is AP? 🔢

An Arithmetic Progression is a sequence where each term increases or decreases by a constant value called the common difference (d).

General Form:
a, (a+d), (a+2d), (a+3d), ...

Where:
• a = first term
• d = common difference
• n = number of terms

Key Formulas for AP 📐

Formula Expression
nth term Tn = a + (n-1)d
Sum of first n terms Sn = n2[2a + (n-1)d]
Sum (alternative) Sn = n2(a + l)
Common difference d = T2 - T1 = T3 - T2

(Where l = last term)

🔢 Example 1: Finding the nth term

Find the 20th term of the AP: 3, 7, 11, 15, ...

Solution:
• First term a = 3
• Common difference d = 7 - 3 = 4
• Using Tn = a + (n-1)d

T20 = 3 + (20-1) × 4
= 3 + 76
= 79
Answer: 79

🔢 Example 2: Finding number of terms

How many terms are in the AP: 5, 9, 13, ..., 101?

Solution:
• a = 5, d = 4, Tn = 101
• Using Tn = a + (n-1)d

101 = 5 + (n-1) × 4
96 = (n-1) × 4
n-1 = 24
n = 25
Answer: 25 terms

🔢 Example 3: Finding sum of AP

Find the sum of the first 15 terms of the AP: 2, 6, 10, 14, ...

Solution:
• a = 2, d = 4, n = 15
• Using Sn = n2[2a + (n-1)d]

S15 = 152[2(2) + (15-1) × 4]
= 152[4 + 56]
= 152 × 60
= 15 × 30
= 450
Answer: 450

🔢 Example 4: Finding a and d

The 3rd term of an AP is 12 and the 7th term is 28. Find the first term and common difference.

Solution:
• T3 = a + 2d = 12 ... (1)
• T7 = a + 6d = 28 ... (2)

Subtract (1) from (2):
4d = 16
d = 4

Substitute into (1):
a + 2(4) = 12
a = 4
Answer: a = 4, d = 4
2️⃣ GEOMETRIC PROGRESSION (GP)

What is GP? 🔄

A Geometric Progression is a sequence where each term is multiplied by a constant value called the common ratio (r).

General Form:
a, ar, ar², ar³, ...

Where:
• a = first term
• r = common ratio

Key Formulas for GP 📐

Formula Expression
nth term Tn = arn-1
Sum of n terms (r > 1) Sn = a(rn - 1)r - 1
Sum of n terms (r < 1) Sn = a(1 - rn)1 - r
Sum to infinity S∞ = a1-r (only when |r| < 1)
Common ratio r = T2T1 = T3T2
Geometric Mean G = √ab

🔢 Example 5: Finding the nth term (GP)

Find the 6th term of the GP: 3, 6, 12, 24, ...

Solution:
• First term a = 3
• Common ratio r = 63 = 2
• Using Tn = arn-1

T6 = 3 × 26-1
= 3 × 25
= 3 × 32
= 96
Answer: 96

🔢 Example 6: Finding sum of GP

Find the sum of the first 8 terms of the GP: 1, 3, 9, 27, ...

Solution:
• a = 1, r = 3, n = 8
• Since r > 1, use Sn = a(rn - 1)r - 1

S8 = 1(38 - 1)3 - 1
= 6561 - 12
= 65602
= 3280
Answer: 3280

🔢 Example 7: Sum to infinity

Find the sum to infinity of the GP: 8, 4, 2, 1, ...

Solution:
• a = 8, r = 48 = 12
• Since |r| = 12 < 1, sum to infinity exists

S∞ = a1-r
= 81-12
= 812
= 16
Answer: 16

🔢 Example 8: Finding a and r

The 2nd term of a GP is 6 and the 5th term is 48. Find the first term and common ratio.

Solution:
• T2 = ar = 6 ... (1)
• T5 = ar4 = 48 ... (2)

Divide (2) by (1):
ar4ar = 486
r3 = 8
r = 2

Substitute into (1):
a × 2 = 6
a = 3
Answer: a = 3, r = 2
3️⃣ SPECIAL TOPICS

3.1 Arithmetic Mean (AM) ➕

If a, A, b are in AP, then:
A = a + b2

3.2 Geometric Mean (GM) ✖️

If a, G, b are in GP, then:
G = √ab

3.3 Relationship between AM and GM ⚖️

For any two positive numbers:
AM ≥ GM

a+b2 ≥ √ab

🎯 JAMB Exam Tips

Common Question Types:

  1. Find the nth term → Use Tn formulas
  2. Find the sum → Use Sn formulas
  3. Given two terms, find a and d (or r) → Set up simultaneous equations
  4. Application problems → Identify if AP or GP from context

⚡ Key Things to Remember

✅ In AP, the difference is constant (add/subtract)

✅ In GP, the ratio is constant (multiply/divide)

✅ For sum to infinity in GP, must check that |r| < 1

✅ When r > 1, use (rn - 1) form; when r < 1, use (1 - rn) form

✅ Always identify a, d (or r), and n before applying formulas

📝 Practice Questions (Try These!)

Q1: Find the sum of the first 20 terms of the AP: 5, 9, 13, ...

Answer: 860

Q2: The sum of the first n terms of an AP is Sn = n² + 3n. Find the 10th term.

Answer: 22

Q3: Find the sum to infinity of the GP: 27, 9, 3, 1, ...

Answer: 40.5

Q4: If the 2nd and 5th terms of a GP are 6 and 162 respectively, find the GP.

Answer: 2, 6, 18, 54, ...

Quick Checklist for Progression Problems:

  • Identify if sequence is AP (common difference) or GP (common ratio)
  • Find first term (a) and common difference/ratio (d or r)
  • For AP: use Tn = a + (n-1)d
  • For GP: use Tn = arn-1
  • For sum: check if r > 1 or r < 1 (for GP)
  • For sum to infinity: verify |r| < 1 first
  • Given two terms: set up simultaneous equations
