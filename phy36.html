📚 Chapter 36: Simple A.C. Circuits

🎯 Learning Objectives: Master alternating current concepts, RMS values, reactance, impedance, power in A.C. circuits, resonance, and solve complex circuit calculations for JAMB success.

1️⃣ Understanding Alternating Current (A.C.) 🔄

Alternating Current (A.C.) is current that periodically reverses direction, flowing first one way then the other, unlike Direct Current (D.C.) which flows in one direction only.

Sinusoidal Waveform:
I = I₀ sin(ωt) or V = V₀ sin(ωt)

Where:
• I₀, V₀ = Peak/Maximum values
• ω = 2πf = Angular frequency (rad/s)
• f = Frequency (Hz)
• t = Time (s)

2️⃣ Values in Alternating Current 📊

Value Definition Formula When to Use
Peak Value Maximum value reached I₀, V₀ (given) Instantaneous calculations
Instantaneous Value at any moment I = I₀ sin(ωt) Specific time values
Mean Value Average over half cycle Iₐᵥ = 2I₀/π ≈ 0.637I₀ Rarely used
RMS Value Effective D.C. equivalent Iᵣₘₛ = I₀/√2 ≈ 0.707I₀ Power calculations
💡 Memory Tip: RMS = "Really Means Something" (the effective value that produces same heating as D.C.)

🔢 Calculation 1: RMS and Peak Values

Question: An A.C. voltage has peak value of 311V. Find the RMS voltage.

Vᵣₘₛ = V₀/√2
= 311/1.414
= 220V
Answer: 220V

3️⃣ Components in A.C. Circuits 🔧

A. Resistor (R) in A.C. Circuit

⚡ Characteristics

  • Phase: Voltage and current are IN PHASE (0° difference)
  • Opposition: Resistance (R) - same as D.C.
  • Formula: Vᵣ = IR
  • Power: Dissipated as heat

B. Inductor (L) in A.C. Circuit

🔄 Characteristics

  • Phase: Current LAGS voltage by 90° (π/2 rad)
  • Opposition: Inductive Reactance (Xₗ)
  • Formula: Xₗ = 2πfL = ωL
  • Key Point: Higher frequency → Higher Xₗ (blocks high frequencies)
  • Power: No dissipation (energy stored in magnetic field)
Memory Aid: "L for Lag - current comes Late"

🔢 Calculation 2: Inductive Reactance

Question: A 0.5H inductor is connected to 50Hz A.C. supply. Calculate Xₗ.

Xₗ = 2πfL
= 2 × 3.142 × 50 × 0.5
= 157.1Ω
Answer: 157.1Ω

C. Capacitor (C) in A.C. Circuit

🔋 Characteristics

  • Phase: Current LEADS voltage by 90° (π/2 rad)
  • Opposition: Capacitive Reactance (Xᴄ)
  • Formula: Xᴄ = 1/(2πfC) = 1/(ωC)
  • Key Point: Higher frequency → Lower Xᴄ (passes high frequencies)
  • Power: No dissipation (energy stored in electric field)
Memory Aid: "C for Current Comes first"

🔢 Calculation 3: Capacitive Reactance

Question: A 10μF capacitor is connected to 50Hz supply. Find Xᴄ.

Xᴄ = 1/(2πfC)
= 1/(2 × 3.142 × 50 × 10 × 10⁻⁶)
= 1/0.003142
= 318.3Ω
Answer: 318.3Ω

4️⃣ Impedance (Z) in A.C. Circuits ⚡

Impedance is the total opposition to current flow in A.C. circuits, combining resistance and reactance.

Series R-L Circuit:
Z = √(R² + Xₗ²)
tan φ = Xₗ/R (Current lags voltage)

Series R-C Circuit:
Z = √(R² + Xᴄ²)
tan φ = Xᴄ/R (Current leads voltage)

Series R-L-C Circuit (Most Important):
Z = √(R² + (Xₗ - Xᴄ)²)
tan φ = (Xₗ - Xᴄ)/R

🎯 Phase Angle Rules

  • Xₗ > Xᴄ: Circuit is INDUCTIVE (current lags) - φ is positive
  • Xᴄ > Xₗ: Circuit is CAPACITIVE (current leads) - φ is negative
  • Xₗ = Xᴄ: Circuit is RESISTIVE (resonance) - φ = 0

🔢 Calculation 4: RLC Circuit Impedance

Question: A series R-L-C circuit has R = 10Ω, L = 0.1H, C = 100μF at 50Hz. Find impedance.

Step 1: Calculate Xₗ
Xₗ = 2π × 50 × 0.1 = 31.42Ω

Step 2: Calculate Xᴄ
Xᴄ = 1/(2π × 50 × 100 × 10⁻⁶) = 31.83Ω

Step 3: Calculate Z
Z = √(10² + (31.42 - 31.83)²)
= √(100 + (-0.41)²)
= √100.17 ≈ 10Ω
Answer: 10Ω (Near resonance)

5️⃣ Power in A.C. Circuits 💡

Power Type Symbol Formula Meaning
Instantaneous p p = vi Power at any instant
Real Power P P = VI cos φ = I²R Actual power consumed (W)
Apparent Power S S = VI Total power supplied (VA)
Reactive Power Q Q = VI sin φ Oscillating power (VAR)
Power Factor:
cos φ = R/Z = Real Power/Apparent Power

• Range: 0 to 1
• Unity (cos φ = 1): All power is real power (at resonance)
• Low power factor: More reactive power, less efficient

🔢 Calculation 5: Power Calculations

Question: A circuit draws 2A from 100V supply with power factor 0.8. Calculate real power.

P = VI cos φ
= 100 × 2 × 0.8
= 160W
Answer: 160W

6️⃣ Resonance in Series R-L-C Circuits 🎯

Resonance occurs when Xₗ = Xᴄ, making the circuit purely resistive.

Resonant Frequency:
f₀ = 1/(2π√(LC))

At Resonance:
• Impedance is minimum: Z = R
• Current is maximum: Iₘₐₓ = V/R
• Power factor = 1 (unity)
• Phase angle φ = 0

🔢 Calculation 6: Resonance Frequency

Question: Find the resonant frequency for L = 1H and C = 10μF.

f₀ = 1/(2π√(LC))
= 1/(2π√(1 × 10 × 10⁻⁶))
= 1/(2π × 3.162 × 10⁻³)
= 1/0.01987
= 50.3Hz
Answer: 50.3Hz

🎯 JAMB Exam Tips

  1. Always convert units: μF to F (×10⁻⁶), mH to H (×10⁻³)
  2. Remember RMS is standard: Unless stated as "peak," assume values are RMS
  3. Watch for resonance: When Xₗ = Xᴄ, calculations become simple (Z = R)
  4. Power factor is always positive between 0 and 1, but phase angle can be positive (lag) or negative (lead)
  5. Draw vector diagrams mentally: R on x-axis, Xₗ up, Xᴄ down
  6. Resistors are Reliable: No phase shift in purely resistive circuits

🔢 Comprehensive JAMB Practice Question

Question: A series circuit consists of R = 30Ω, L = 0.4H, and C = 20μF connected to 100V, 50Hz supply. Calculate: (a) Xₗ, (b) Xᴄ, (c) Z, (d) Current, (e) Phase angle, (f) Power factor.

(a) Xₗ = 2πfL = 2 × 3.142 × 50 × 0.4 = 125.7Ω

(b) Xᴄ = 1/(2πfC) = 1/(2 × 3.142 × 50 × 20 × 10⁻⁶) = 159.2Ω

(c) Z = √(R² + (Xₗ-Xᴄ)²)
= √(30² + (125.7-159.2)²)
= √(900 + 1122.25) = 45.0Ω

(d) I = V/Z = 100/45 = 2.22A

(e) tan φ = (Xₗ-Xᴄ)/R = -33.5/30 = -1.117
φ = tan⁻¹(-1.117) = -48.1° (Capacitive - current leads)

(f) cos φ = R/Z = 30/45 = 0.667
Answers: (a) 125.7Ω (b) 159.2Ω (c) 45Ω (d) 2.22A (e) -48.1° (f) 0.667
