1️⃣ NUMBER BASES

What are Number Bases? 🤔

A number base (or radix) is the number of unique digits—including zero—used to represent numbers in a positional numeral system.

Base Name Digits Used
2 Binary 0, 1
8 Octal 0-7
10 Decimal 0-9
16 Hexadecimal 0-9, A-F

Converting from Other Bases to Base 10

⚡ Method: Expand using powers of the base

Multiply each digit by the base raised to its position power (starting from 0 on the right).

🔢 Example 1: Convert 1011₂ to base 10

Solution:
1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰
= 8 + 0 + 2 + 1
= 11₁₀
Answer: 11₁₀

🔢 Example 2: Convert 243₅ to base 10

Solution:
243₅ = 2×5² + 4×5¹ + 3×5⁰
= 2×25 + 4×5 + 3×1
= 50 + 20 + 3
= 73₁₀
Answer: 73₁₀

Converting from Base 10 to Other Bases

⚡ Method: Repeated Division

Divide by the new base repeatedly and collect remainders in reverse order.

🔢 Example 3: Convert 45₁₀ to base 2

Solution:
Division Quotient Remainder
45 ÷ 2221
22 ÷ 2110
11 ÷ 251
5 ÷ 221
2 ÷ 210
1 ÷ 201

Reading remainders ↑ upwards: 101101
Answer: 101101₂

🔢 Example 4: Convert 200₁₀ to base 5

Solution:
Division Quotient Remainder
200 ÷ 5400
40 ÷ 580
8 ÷ 513
1 ÷ 501

Reading remainders ↑ upwards: 1300
Answer: 1300₅

Converting Between Non-Decimal Bases

💡 Remember: Base 10 is the bridge! Convert to base 10 first, then to target base.

🔢 Example 5: Convert 101₂ to base 4

Solution:
Step 1: Convert to base 10
101₂ = 1×4 + 0×2 + 1 = 5₁₀

Step 2: Convert 5₁₀ to base 4
• 5 ÷ 4 = 1 remainder 1
• 1 ÷ 4 = 0 remainder 1

5₁₀ = 11₄
Answer: 11₄

🚀 Shortcut: Binary to Base 4 (2²)

Group binary digits in pairs from right: 101₂ → 01 01 (pad with zero) → 1 1 = 11₄

2️⃣ FRACTIONS, DECIMALS & APPROXIMATIONS

Types of Fractions 📊

  • Proper fraction: Numerator < Denominator (e.g., 34)
  • Improper fraction: Numerator ≥ Denominator (e.g., 74)
  • Mixed number: Whole number + proper fraction (e.g., 134)

Converting Recurring Decimals to Fractions

🔢 Example 6: Convert 0.3̇ to a fraction

Solution:
Let x = 0.333...
10x = 3.333...

Subtract: 10x - x = 3.333... - 0.333...
9x = 3
x = 39 = 13
Answer: ⅓

🔢 Example 7: Convert 0.16̇ to a fraction

Solution:
Let x = 0.1666...
10x = 1.666... and 100x = 16.666...

Subtract: 100x - 10x = 16.666... - 1.666...
90x = 15
x = 1590 = 16
Answer: ⅙

Percentage Error

Percentage Error =
|Exact Value - Approximate Value| Exact Value × 100%

🔢 Example 8: Calculate Percentage Error

A length is measured as 5.2cm instead of the true value 5.15cm.

Solution:
Error = |5.15 - 5.2|5.15 × 100%
= 0.055.15 × 100%
≈ 0.97%
Answer: ≈ 0.97%
3️⃣ INDICES AND LOGARITHMS

Laws of Indices 📈

Law Formula Example
Multiplication aᵐ × aⁿ = aᵐ⁺ⁿ 2³ × 2⁴ = 2⁷ = 128
Division aᵐ ÷ aⁿ = aᵐ⁻ⁿ 5⁶ ÷ 5² = 5⁴ = 625
Power of power (aᵐ)ⁿ = aᵐⁿ (3²)³ = 3⁶ = 729
Zero index a⁰ = 1 (a ≠ 0) 7⁰ = 1
Negative index a⁻ⁿ = 1/aⁿ 2⁻³ = ⅛
Fractional index a^(m/n) = ⁿ√(aᵐ) 8^(⅔) = (³√8)² = 4

🔢 Example 9: Simplify (3⁵ × 9²)/27³

Solution:
Express all with base 3:
• 9 = 3², so 9² = (3²)² = 3⁴
• 27 = 3³, so 27³ = (3³)³ = 3⁹

3⁵ × 3⁴3⁹ = 3⁹3⁹ = 3⁰ = 1
Answer: 1

Logarithms

If aˣ = N, then logₐ N = x

Laws of Logarithms

  • • logₐ(MN) = logₐ M + logₐ N
  • • logₐ(M/N) = logₐ M - logₐ N
  • • logₐ(Mⁿ) = n·logₐ M

🔢 Example 10: Solve log₂ x + log₂(x-2) = 3

Solution:
log₂[x(x-2)] = 3
x(x-2) = 2³ = 8
x² - 2x - 8 = 0
(x-4)(x+2) = 0

x = 4 or x = -2

Since log of negative numbers is undefined, x = 4
Answer: x = 4
4️⃣ SURDS

What are Surds? √

Surds are irrational numbers expressed as roots (e.g., √2, ³√5).

Simplifying Surds

🔢 Example 11: Simplify √50

Solution:
√50 = √(25 × 2) = √25 × √2 = 5√2
Answer: 5√2

Rationalizing the Denominator

🔢 Example 12: Rationalize 3/√5

Solution:
3√5 × √5√5 = 3√55
Answer: 3√5 / 5

🔢 Example 13: Rationalize 2/(3-√2)

Solution:
Multiply by conjugate (3+√2):

23-√2 × 3+√23+√2 = 2(3+√2)9-2 = 6+2√27
Answer: (6 + 2√2) / 7
5️⃣ SETS

Set Notation { }

  • ∈ : "is an element of"
  • ⊂ : "is a subset of"
  • ∩ : Intersection (AND)
  • ∪ : Union (OR)
  • A' or Aᶜ : Complement of A
  • n(A) : Number of elements in A

🔢 Example 14: Venn Diagram Problem

In a class of 40 students: 20 study Mathematics, 18 study Physics, 8 study both subjects. Find how many study neither.

Solution:
• Only Math: 20 - 8 = 12
• Only Physics: 18 - 8 = 10
• Both: 8
• Total in sets: 12 + 10 + 8 = 30
• Neither: 40 - 30 = 10
Answer: 10 students
6️⃣ NUMBER SYSTEMS

Types of Numbers 🔢

Integers ℤ

Whole numbers including negatives: {..., -2, -1, 0, 1, 2, ...}

Rational Numbers ℚ

Can be expressed as p/q where p, q are integers, q ≠ 0
Examples: ¾, -2, 0.75, 0.333...

Irrational Numbers

Cannot be expressed as simple fractions
Examples: √2, π, e

Real Numbers ℝ

All rational and irrational numbers combined.

🎯 JAMB Exam Tips

  1. Master base conversions — these appear every year
  2. Remember indices laws — especially negative and fractional indices
  3. Logarithms — practice solving equations using log properties
  4. Surds — always rationalize denominators in final answers
  5. Sets — draw Venn diagrams for visualization
  6. Time management — these questions should take 1-2 minutes each

Quick Checklist for Number Problems:

  • Identify the number base first
  • Check for shortcuts (grouping for binary conversions)
  • Remember: Base 10 is always the bridge
  • For indices: express all terms with same base
  • For surds: look for perfect square factors
  • For sets: draw Venn diagram before calculating
