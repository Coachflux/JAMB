📚 What is a Locus?

A locus (plural: loci) is the path traced out by a point that moves according to a specific rule or set of conditions.

💡 Simple Definition: A locus is the set of all points that satisfy a given geometric condition.

1️⃣ EQUIDISTANT FROM TWO POINTS

Perpendicular Bisector 📏

Rule: The perpendicular bisector of the line joining the two points.

If P(x,y) is equidistant from A(x₁,y₁) and B(x₂,y₂):
√[(x-x₁)²+(y-y₁)²] = √[(x-x₂)²+(y-y₂)²]

🔢 Example 1: Two Points

Find the locus of P(x,y) equidistant from A(2,3) and B(6,7).

PA = PB

√[(x-2)²+(y-3)²] = √[(x-6)²+(y-7)²]

Squaring both sides:
(x-2)²+(y-3)² = (x-6)²+(y-7)²

Expanding:
x²-4x+4+y²-6y+9 = x²-12x+36+y²-14y+49

Simplifying:
-4x-6y+13 = -12x-14y+85
8x+8y = 72
x + y = 9

(Straight line - perpendicular bisector of AB)

2️⃣ EQUIDISTANT FROM TWO LINES

Angle Bisectors 📐

Rule: The angle bisectors of the angles formed by the two lines.

Distance from point (x,y) to line ax+by+c=0:
d = |ax+by+c|/√(a²+b²)

🔢 Example 2: Two Intersecting Lines

Find locus equidistant from 3x+4y-7=0 and 5x-12y+3=0.

|3x+4y-7|/√(3²+4²) = |5x-12y+3|/√(5²+12²)

|3x+4y-7|/5 = |5x-12y+3|/13

Case 1: 13(3x+4y-7) = 5(5x-12y+3)
39x+52y-91 = 25x-60y+15
14x+112y-106 = 0
7x+56y-53 = 0

Case 2: 13(3x+4y-7) = -5(5x-12y+3)
39x+52y-91 = -25x+60y-15
64x-8y-76 = 0
16x-2y-19 = 0
3️⃣ CONSTANT DISTANCE FROM POINT

Circle ⭕

Rule: A circle with the fixed point as center and constant distance as radius.

Equation: (x-a)² + (y-b)² = r²
Center: (a,b), Radius: r

🔢 Example 3: Circle Locus

Find locus of point always 5 units from (2,-3).

(x-2)² + (y-(-3))² = 5²
(x-2)² + (y+3)² = 25

Expanding:
x²-4x+4+y²+6y+9 = 25
x²+y²-4x+6y+13 = 25
x² + y² - 4x + 6y - 12 = 0
4️⃣ CONSTANT DISTANCE FROM LINE

Parallel Lines 🔄

Rule: Two parallel lines, one on each side of the given line.

Distance d from line ax+by+c=0 gives two lines:
ax+by+c = ±d√(a²+b²)

🔢 Example 4: Parallel Lines

Find locus of points at distance 3 units from line 4x-3y+5=0.

|4x-3y+5|/√(4²+(-3)²) = 3
|4x-3y+5|/5 = 3
|4x-3y+5| = 15

Case 1: 4x-3y+5 = 15
4x-3y-10 = 0

Case 2: 4x-3y+5 = -15
4x-3y+20 = 0
5️⃣ EQUIDISTANT FROM POINT & LINE

Parabola 📈

💡 This is the geometric definition of a parabola: locus equidistant from focus (point) and directrix (line).

🔢 Example 5: Parabola

Find locus equidistant from point (0,3) and line y=-3.

Distance to (0,3) = Distance to y=-3

√[(x-0)²+(y-3)²] = |y-(-3)|
√[x²+(y-3)²] = |y+3|

Squaring:
x²+(y-3)² = (y+3)²
x²+y²-6y+9 = y²+6y+9
x² = 12y
x² = 12y

(Parabola opening upward, vertex at origin)

📝 JAMB PRACTICE QUESTIONS

Worked Examples 🎯

Question 1 (Basic):

Find locus equidistant from A(1,2) and B(5,6).

√[(x-1)²+(y-2)²] = √[(x-5)²+(y-6)²]

(x-1)²+(y-2)² = (x-5)²+(y-6)²
x²-2x+1+y²-4y+4 = x²-10x+25+y²-12y+36
8x+8y = 56
x + y = 7

Question 2 (Intermediate):

Point P moves so its distance from origin is twice its distance from (3,0). Find the locus.

PO = 2 × P(3,0)
√[x²+y²] = 2√[(x-3)²+y²]

x²+y² = 4[(x-3)²+y²]
x²+y² = 4x²-24x+36+4y²
0 = 3x²-24x+36+3y²
x²-8x+12+y² = 0
(x-4)²+y² = 4
Circle: center (4,0), radius 2

Question 3 (Advanced - Ellipse):

Find locus where sum of distances from (-3,0) and (3,0) is always 10.

This is an ellipse definition:
Foci at (±3,0), so c = 3
Major axis 2a = 10, so a = 5

b² = a²-c² = 25-9 = 16
b = 4
x²25 + y²16 = 1

💡 Key Tips for JAMB

1. Always sketch: Draw diagram to visualize

2. Use distance formula: √[(x₂-x₁)²+(y₂-y₁)²]

3. Simplify carefully: Expand and collect like terms

4. Check your answer: Test a point on your locus

5. Know standard forms: Circle, Parabola, Ellipse, Hyperbola

⚡ Common JAMB Patterns

🔹 "Equidistant from two points" → Perpendicular bisector

🔹 "Constant distance from point" → Circle

🔹 "Sum of distances constant" → Ellipse

🔹 "Difference of distances constant" → Hyperbola

🔹 "Equidistant from point and line" → Parabola

🔹 "Constant distance from line" → Parallel lines
