📍 Chapter 20: Measures of Location

This chapter covers mean, median, mode, quartiles and percentiles - fundamental statistics for finding central values in data sets.

1️⃣ MEAN (ARITHMETIC MEAN)

The Average 📐

Formula:
x̄ = ∑ xn

Where:
• ∑x = sum of all values
• n = number of values

🔢 Example 1: Simple Mean

Find the mean of: 5, 8, 12, 15, 20

x̄ = 5 + 8 + 12 + 15 + 205
= 605
= 12
Answer: Mean = 12

🔢 Example 2: Mean from Frequency Table

Score (x) Frequency (f) f × x
2 3 6
4 5 20
6 2 12
Total ∑f = 10 ∑fx = 38
x̄ = ∑fx∑f = 3810 = 3.8
Answer: Mean = 3.8
💡 Mean = Sum of everything ÷ How many things
2️⃣ MEDIAN

The Middle Value 📊

For Ungrouped Data:
• Odd count: Middle value directly
• Even count: Average of two middle values

For Grouped Data:
Median = L + n2 − Fbfm × c

Where:
• L = lower class boundary of median class
• n = total frequency
• Fb = cumulative frequency before median class
• fm = frequency of median class
• c = class width

🔢 Example 3: Median (Ungrouped - Odd)

Data: 3, 7, 8, 12, 15

Data arranged: 3, 7, 8, 12, 15
Middle value (3rd) = 8
Answer: Median = 8

🔢 Example 4: Median (Ungrouped - Even)

Data: 4, 6, 10, 14

Middle values: 6 and 10
Median = 6 + 102 = 162 = 8
Answer: Median = 8

🔢 Example 5: Median from Grouped Data

Class Frequency (f) Cumulative Frequency
10-19 5 5
20-29 8 13
30-39 12 25
40-49 10 35
50-59 5 40
Step 1: n = 40, so n2 = 20

Step 2: Median class = 30-39 (where cumulative freq reaches 20)

Step 3: Values:
• L = 29.5 (lower boundary)
• Fb = 13
• fm = 12
• c = 10

Step 4: Calculate
Median = 29.5 + 20 − 1312 × 10
= 29.5 + 712 × 10
= 29.5 + 5.83
= 35.33
Answer: Median = 35.33
3️⃣ MODE

Most Frequent Value 📈

For Ungrouped Data:
Value that appears most frequently

For Grouped Data:
Mode = L + D1D1 + D2 × c

Where:
• L = lower boundary of modal class (highest frequency)
• D1 = modal freq − previous freq
• D2 = modal freq − next freq
• c = class width

🔢 Example 6: Mode (Ungrouped)

Data: 2, 3, 3, 5, 7, 3, 8

• 2 appears 1 time
• 3 appears 3 times ← Most frequent
• 5 appears 1 time
• 7 appears 1 time
• 8 appears 1 time
Answer: Mode = 3

🔢 Example 7: Mode from Grouped Data

Using the same table (Example 5):

Modal class: 30-39 (highest frequency = 12)

Values:
• L = 29.5
• D1 = 12 − 8 = 4
• D2 = 12 − 10 = 2
• c = 10

Mode = 29.5 + 44 + 2 × 10
= 29.5 + 46 × 10
= 29.5 + 6.67
= 36.17
Answer: Mode = 36.17
4️⃣ PERCENTILES & QUARTILES

Data Division Points 📏

Quartiles (Q):

• Q₁ (1st quartile) = 25th percentile

• Q₂ (2nd quartile) = Median = 50th percentile

• Q₃ (3rd quartile) = 75th percentile

Formula for Quartiles (Grouped Data):
Qk = L + kn4 − Fbf × c

Where k = 1, 2, 3 for Q₁, Q₂, Q₃

Percentiles:
P = L + Pn100 − Fbf × c

🔢 Example 8: Finding Q₁

Using grouped data from Example 5 (n = 40):

Step 1: Position = 1 × 404 = 10

Step 2: Q₁ class = 20-29 (cumulative freq 13 covers position 10)

Values:
• L = 19.5
• Fb = 5
• f = 8
• c = 10

Q₁ = 19.5 + 10 − 58 × 10
= 19.5 + 58 × 10
= 19.5 + 6.25
= 25.75
Answer: Q₁ = 25.75

⚡ Key Variables Quick Guide

L: Lower boundary (subtract 0.5 from lower limit)

n: Total frequency (sum of all f)

Fb: Cumulative frequency BEFORE the class

c: Class width (upper − lower boundary)

D1, D2: Differences with adjacent classes

✏️ PRACTICE QUESTION:

Ages of 50 students:

Age (years): 10-14, 15-19, 20-24, 25-29, 30-34

Frequency: 5, 12, 18, 10, 5

Find: (a) Mean (b) Median (c) Mode

Answers: Mean = 21.8, Median = 21.72, Mode = 21.64

🎯 JAMB EXAM TIPS

1. Check data type: Grouped or ungrouped determines formula

2. Class boundaries: 10-19 → 9.5-19.5 (subtract 0.5)

3. Cumulative frequency: Calculate first for median/quartiles

4. Modal class: Highest frequency (not the mode itself)

5. Symmetry check: Mean = Mode = Median for symmetrical distribution

⚠️ Common Mistakes:

• Using class limits instead of boundaries (always use 9.5, not 10)

• Forgetting to calculate cumulative frequency first

• Confusing modal class with the actual mode value

• Wrong position calculation for quartiles (use kn/4, not n/4)
