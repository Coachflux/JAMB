What is a Matrix? 🤔

A matrix is a rectangular array of numbers arranged in rows and columns, enclosed in brackets.

Example:
A = ( 1 2 3 )
     ( 4 5 6 )

This is a 2×3 matrix (2 rows, 3 columns)

2️⃣ ALGEBRA OF MATRICES

A. Addition & Subtraction ➕➖

💡 Rule: Matrices must have the same order (same number of rows and columns). Add/subtract corresponding elements.

🔢 Example 1: Addition

( 2 5 ) + ( 4 2 ) = ( 2+4 5+2 ) = ( 6 7 )
( 3 1 )   ( 1 6 )   ( 3+1 1+6 )   ( 4 7 )

🔢 Example 2: Subtraction

( 8 3 ) - ( 2 1 ) = ( 6 2 )
( 5 9 )   ( 4 3 )   ( 1 6 )

B. Scalar Multiplication ✖️

Multiply every element by the scalar (number).

🔢 Example 3: Scalar Multiplication

3 × ( 2 4 ) = ( 6 12 )
      ( 1 5 )   ( 3 15 )

C. Matrix Multiplication ⚠️ VERY IMPORTANT

⚠️ Key Rules:

• Number of columns in A must equal number of rows in B

• Result has dimensions: (rows of A) × (columns of B)

• Method: Multiply row by column and sum the products

🔢 Example 4: 2×2 by 2×2

( 1 2 ) × ( 5 6 ) = ?

( 3 4 )   ( 7 8 )

Step-by-step:

Row 1, Column 1: (1×5) + (2×7) = 5 + 14 = 19
Row 1, Column 2: (1×6) + (2×8) = 6 + 16 = 22
Row 2, Column 1: (3×5) + (4×7) = 15 + 28 = 43
Row 2, Column 2: (3×6) + (4×8) = 18 + 32 = 50

Result: ( 19 22 )
          ( 43 50 )

🔢 Example 5: 2×3 by 3×2

3️⃣ DETERMINANTS

What is a Determinant? 🔢

The determinant is a special number calculated from a square matrix. Denoted as |A| or det(A).

A. Determinant of 2×2 Matrix 📐

For A = ( a b )
          ( c d )

|A| = ad - bc

🔢 Example 6: 2×2 Determinant

| 3 5 | = (3×4) - (5×2) = 12 - 10 = 2
| 2 4 |

B. Determinant of 3×3 Matrix 🧮

For A = ( a b c )
          ( d e f )
          ( g h i )

|A| = a(ei - fh) - b(di - fg) + c(dh - eg)

Memory Tip: Expand along first row using alternating signs (+, −, +)

🔢 Example 7: 3×3 Determinant

| 1 2 3 |
| 4 5 6 |
| 7 8 9 |

= 1| 5 6 | - 2| 4 6 | + 3| 4 5 |
     | 8 9 |     | 7 9 |     | 7 8 |

= 1(45-48) - 2(36-42) + 3(32-35)
= 1(-3) - 2(-6) + 3(-3)
= -3 + 12 - 9
= 0
4️⃣ INVERSE OF A MATRIX
💡 Only square matrices with non-zero determinants have inverses.

Inverse of 2×2 Matrix ⭐ JAMB FAVORITE

For A = ( a b )
          ( c d )

A-1 = 1|A| ( d -b )
                       ( -c a )

Steps:
1. Find determinant |A| = ad - bc
2. Swap a and d (main diagonal)
3. Change signs of b and c (off-diagonal)
4. Multiply by 1|A|

🔢 Example 8: Finding Inverse

Find the inverse of A = ( 4 7 )

                                ( 2 6 )

Step 1: |A| = (4×6) - (7×2) = 24 - 14 = 10

Step 2: Apply formula
A-1 = 110 ( 6 -7 )
                     ( -2 4 )

= ( 0.6 -0.7 )
    ( -0.2 0.4 )

Or in fractions:
= ( 35 -710 )
    ( -15 25 )

Using Inverse to Solve Simultaneous Equations 📐

🔢 Example 9: Solving with Matrices

Solve:

2x + 3y = 8

5x + 4y = 13

Matrix Form: AX = B
( 2 3 ) ( x ) = ( 8 )
( 5 4 ) ( y )   ( 13 )

Solution: X = A-1B

Step 1: |A| = (2×4) - (3×5) = 8 - 15 = -7

Step 2: A-1 = 1-7( 4 -3 ) = ( -47 37 )
                         ( -5 2 )   ( 57 -27 )

Step 3: Multiply A-1 × B
( x ) = ( -47×8 + 37×13 ) = ( -32+397 ) = ( 1 )
( y )   ( 57×8 + (-27)×13 )   ( 40-267 )   ( 2 )
Answer: x = 1, y = 2

🎯 JAMB Past Question Style

Likely Question:

"If A = ( 2 1 ) and B = ( 1 2 ), find AB and the determinant of AB."

            ( 3 4 )          ( 0 3 )

Solution:
1. AB = ( (2×1+1×0) (2×2+1×3) ) = ( 2 7 )
           ( (3×1+4×0) (3×2+4×3) )   ( 3 18 )

2. |AB| = (2×18) - (7×3) = 36 - 21 = 15

⚡ Key Points to Remember

✅ Matrix multiplication is NOT commutative: AB ≠ BA (usually)

✅ Determinant of identity matrix = 1

✅ If determinant = 0, matrix has NO inverse (singular matrix)

✅ For inverse: Swap diagonal, negate off-diagonal, divide by determinant

✅ Always check matrix orders before attempting operations

✅ 3×3 determinants: Use cofactor expansion (alternating signs +, −, +)

Quick Checklist for Matrix Problems:

  • Check matrix dimensions before operations
  • For multiplication: verify columns of A = rows of B
  • For determinants: matrix must be square
  • For inverse: check |A| ≠ 0 first
  • Remember: swap diagonal, negate off-diagonal for 2×2 inverse
  • For 3×3 determinants: use alternating signs (+, −, +)
  • Solving equations: X = A-1B (pre-multiply)
