📚 Chapter 6: Variation

🎯 Learning Objectives: Solve problems involving direct, inverse, joint, partial, and combined variation, including percentage changes.

1️⃣ DIRECT VARIATION

Concept ➡️

When one variable increases, the other increases at the same rate.

💡 Key phrases: "y varies directly as x" or "y is directly proportional to x"
Formula:
y = kx   or   y₁x₁ = y₂x₂

Where k = constant of proportionality

🔢 Example 1: Basic Direct Variation

y varies directly as x. When y = 20, x = 2. Find y when x = 8.

Method 1 - Formula:
1. Start with y = kx
2. Substitute: 20 = k(2)
3. Solve: k = 10
4. New equation: y = 10x
5. When x = 8: y = 10(8) = 80
Method 2 - Proportion (Faster):
y₁x₁ = y₂x₂
202 = y8
2y = 160
y = 80
Answer: y = 80

🔢 Example 2: Direct Square Variation

y varies directly as the square of x. When y = 4, x = 3. Find y when x = 2.

Solution:
y = kx²
4 = k(3)² ⇒ 4 = 9k ⇒ k = 49

New equation: y = 49x²
When x = 2: y = 49(2)² = 49(4) = 169
Answer: y = 16/9
2️⃣ INVERSE VARIATION

Concept ⬅️➡️

When one variable increases, the other decreases. Their product is constant.

💡 Key phrases: "y varies inversely as x" or "y is inversely proportional to x"
Formula:
y = kx   or   xy = k   or   x₁y₁ = x₂y₂

🔢 Example 3: Inverse Variation

y varies inversely as x. When y = 4, x = 3. Find x when y = 6.

Method 1 - Formula:
1. y = kx
2. 4 = k3 ⇒ k = 12
3. New equation: y = 12x
4. When y = 6: 6 = 12x ⇒ 6x = 12 ⇒ x = 2
Method 2 - Product Rule (Faster):
x₁y₁ = x₂y₂
(3)(4) = x₂(6)
12 = 6x₂
x₂ = 2
Answer: x = 2
3️⃣ JOINT VARIATION

Concept 🔗

One variable varies directly with two or more other variables simultaneously.

💡 Key phrase: "y varies jointly as x and z"
Formula:
y = kxz   or   y = kx√z   (depending on relationship)

🔢 Example 4: Joint Variation

x varies jointly as y and the square root of z. When x = -18, y = 2, and z = 9. Find y when x = 10 and z = 4.

Solution:
x = ky√z
-18 = k(2)√9
-18 = k(2)(3) = 6k
k = -3

New equation: x = -3y√z

When x = 10 and z = 4:
10 = -3y√4
10 = -3y(2) = -6y
y = 10-6 = -53
Answer: y = -5/3
4️⃣ PARTIAL VARIATION

Concept ➕

A relationship consisting of a constant part plus a variable part.

💡 Key phrase: "y is partly constant and partly varies as x"
Formula:
y = k₁x + k₂

Where k₁ = variation constant, k₂ = constant term

🔢 Example 5: Partial Variation

y is partly constant and partly varies directly with x. When y = 4, x = 2, and when y = 16, x = 4. Find y when x = 6.

Solution:
Let y = k₁x + k₂

From first condition: 4 = 2k₁ + k₂ ... (1)
From second condition: 16 = 4k₁ + k₂ ... (2)

Subtract (1) from (2):
12 = 2k₁ ⇒ k₁ = 6

Substitute back into (1):
4 = 2(6) + k₂ ⇒ 4 = 12 + k₂ ⇒ k₂ = -8

Equation: y = 6x - 8

When x = 6:
y = 6(6) - 8 = 36 - 8 = 28
Answer: y = 28
5️⃣ COMBINED VARIATION

Concept ⚡

Mixture of direct, inverse, and joint variations in one relationship.

💡 Key phrase: "y varies jointly as x and w and inversely as the square of z"
Formula:
y = kxwz²

🔢 Example 6: Combined Variation

y varies jointly as x and w and inversely as the square of z. When y = 100, x = 2, w = 4, and z = 20.

(a) Find the equation of variation

(b) Find y when x = 1, w = 5, and z = 4

Solution (a):
y = kxwz²
100 = k(2)(4)(20)² = 8k400
8k = 100 × 400 = 40000
k = 400008 = 5000

Equation: y = 5000xwz²
Solution (b):
y = 5000(1)(5)4² = 2500016 = 1562.5
Answers: (a) y = 5000xw/z² (b) y = 1562.5
6️⃣ PERCENTAGE INCREASE/DECREASE

Concept 📊

Calculate percentage changes when variables in a variation relationship change.

🔢 Example 7: Percentage Change in Variation

The area of a triangle varies jointly with its base and height. If the base is increased by 40% and the height is decreased by 10%, what is the percentage change in the area?

Solution:
Original: A = kbh

New conditions:
• Base becomes 1.4b (40% increase)
• Height becomes 0.9h (10% decrease)

New area:
Anew = k(1.4b)(0.9h) = k(1.26)bh = 1.26(kbh) = 1.26Aoriginal

Change = 1.26 - 1.00 = 0.26 = 26% increase
Answer: 26% increase

🎯 Quick Tips for JAMB Exam

  1. Identify the type first — Look for keywords in the question
  2. Find k first — Always calculate the constant of proportionality using given values
  3. Write the equation — Once you have k, write the complete variation equation
  4. Substitute new values — Use the equation to find unknowns
  5. For partial variation — Set up two simultaneous equations and solve
  6. Check units — Make sure your answer makes sense in context

⚡ Practice Strategy

1. Read the problem carefully and underline variation keywords

2. Write down the appropriate formula template

3. Substitute given values to find k

4. Write the specific equation for this problem

5. Solve for the required unknown

Quick Checklist for Variation Problems:

  • Identify variation type from keywords
  • Write general formula with constant k
  • Substitute given values to find k
  • Write specific equation with found k value
  • For partial variation: use simultaneous equations
  • For percentage changes: use multipliers (1.4 for 40% increase, etc.)
  • Check if answer is reasonable in context
