🧠 Quick memory

∆G = ∆H − T∆S. If ∆S is in J, convert to kJ before using with ∆H in kJ.

1️⃣ Energy Changes (∆H) — Enthalpy

∆H is the heat change at constant pressure.

Type What happens Effect on surroundings Sign of ∆H
Exothermic Heat released Surroundings warmer Negative (−∆H)
Endothermic Heat absorbed Surroundings cooler Positive (+∆H)
← Swipe table to see more →

Quick check: Temperature ↑ → exothermic (−∆H). Temperature ↓ → endothermic (+∆H).

Examples

  • Na(s) + H₂O(l) → NaOH(aq) + ½H₂(g) — exothermic (very hot)
  • NH₄Cl(s) → NH₄⁺(aq) + Cl⁻(aq) — endothermic (cold)
Energy diagrams: exothermic products lower; endothermic products higher

2️⃣ Entropy (∆S) — Order vs Disorder

Entropy measures disorder/randomness. Higher disorder = higher entropy (∆S positive).

When ∆S is usually positive

  • Solid → liquid → gas (disorder increases)
  • Mixing gases (spreading out)
  • Dissolving salts (ions free to move)
🧠 Shortcut

If particles become freer to move, ∆S is likely positive.

3️⃣ Spontaneity — Gibbs Free Energy (∆G)

Formula: ∆G° = ∆H° − T∆S° (T in Kelvin).

∆G value Meaning
∆G < 0 Spontaneous
∆G = 0 Equilibrium
∆G > 0 Non-spontaneous
← Swipe table to see more →

Sign combination table (very important)

∆H ∆S Spontaneity
− (exo) + Always spontaneous
− (exo) − Spontaneous at low T
+ (endo) + Spontaneous at high T
+ (endo) − Never spontaneous

Unit warning

  • If ∆H is in kJ/mol and ∆S in J/mol·K, convert ∆S to kJ/mol·K by dividing by 1000.

🧮 6 JAMB Examples (Worked)

Example 1: Hot/cold beaker

NaOH dissolves and beaker feels hot. Exo or endo?

Temperature of surroundings rises → heat released

So ∆H is negative → exothermic

Example 2: Calculate ∆G and spontaneity

∆H° = −92 kJ/mol, ∆S° = +198 J/mol·K, T = 298 K

Convert ∆S°: 198 J/mol·K = 0.198 kJ/mol·K

T∆S° = 298×0.198 = 59.004 kJ/mol

∆G° = −92 − 59.004 = −151 kJ/mol → spontaneous

Example 3: Temperature for ∆G = 0

∆H° = +178 kJ/mol, ∆S° = +160 J/mol·K. Find T when ∆G = 0

∆S° = 0.160 kJ/mol·K

T = ∆H/∆S = 178/0.160 = 1112.5 K

Above 1112.5 K → spontaneous

Example 4: Entropy from state change

Ice melts to water: ∆S positive or negative?

Solid (more ordered) → liquid (less ordered) → disorder increases

∆S positive

Example 5: Energy diagram interpretation

Products lower energy than reactants: exo or endo?

Energy released to surroundings → ∆H negative

Exothermic

Example 6: ∆G with signs

∆H = +120 kJ/mol, ∆S = −0.15 kJ/mol·K, T = 298 K

T∆S = 298×(−0.15) = −44.7 kJ/mol

∆G = 120 − (−44.7) = 164.7 kJ/mol

∆G > 0 → non-spontaneous

✅ Quick Revision Checklist

  • Exothermic (−∆H) vs endothermic (+∆H) using temperature change
  • Entropy increases with disorder (solid→liquid→gas)
  • ∆G = ∆H − T∆S (T in Kelvin)
  • ∆G sign decides spontaneity
  • 4 sign-combinations table (low T vs high T)
  • Unit conversion: J to kJ for ∆S
🧠 Final memory line

If ∆H is negative and ∆S is positive, you can confidently pick “spontaneous” without calculation.
